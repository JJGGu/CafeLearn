# TANK200D：同机型与同类机型比赛案例

先说结论：

- 公开网络上，能直接核实“前几名明确使用 `TANK200D` / `Tank200` 参赛，并且留下完整曲线”的资料非常少。
- 但能找到三类很有用的材料：
  - `同平台 / 同系列`：`Tank200` 使用者、店铺、比赛备赛记录
  - `同品牌赛事机`：顽固品牌出现在正式赛事设备里
  - `同类小型电热样品机`：世界赛 / 国家赛前几名公开承认长期使用，并愿意分享操作逻辑

对你最有价值的，不是“照抄别人的温度数字”，而是把这些案例里的：

- 装豆量逻辑
- 前段恢复节奏
- 中段降能量方式
- 一爆后怎么手动修正
- 样品烘与正式烘如何对齐

迁移到你的 `TANK200D` 上。

## 先把机器边界说清楚

按你本地资料里的定义，`TANK200D` 是：

- `120g-220g`
- `1400W`
- 小型电热直火滚筒
- `55RPM`
- 没有强响应、可精细独立控制的专业风门系统

这意味着：

1. 你可以学比赛线的“热量结构”，不能直接抄别人的风量数值。
2. 你可以学一爆前后怎么收能量，但不能把 `Bullet`、`ROEST`、`LINK` 的自动控制曲线直接等比例搬过来。
3. 你越想接近比赛风格浅烘，越应该把装豆量压在 `150g-180g` 一档，而不是长期顶着 `200g-220g` 满载练。

## B级资料：不是你的同一台机器，但比赛逻辑和你很接近

### 3. Andrew Coe：`2023 US Roasting Champion`，`2023 WCRC` 世界第 `3`，长期使用 `Aillio Bullet`

公开页面能直接确认：

- `Andrew Coe` 自 `2018` 年起使用 `Bullet`
- 他在 `2023` 拿到美国烘焙冠军
- 同年世界赛拿到第 `3`

更重要的是，`Roast.World` 社区讨论里补出了真正有用的操作信息：

- 他的一个核心思路是：前段高功率配合较高风量，让气流更多参与传热，而不是只靠表面烫
- 对自然处理豆，如果满功率会出现 `tipping / scorching`，可以把最前 `45-60` 秒放缓一点
- 对水洗豆，他自己现在更倾向于不要 `soak`，以免损失系统热量
- 风量不是越大越好，而是找“足够排烟除银皮，但还没把锅吹冷”的上限

### 这条案例怎么迁移到 `TANK200D`

你不能照抄 `Bullet` 的 `F5 / F4 / F3`。

但你可以直接搬这四个原则：

1. 水洗比赛线，不要把前段做成又慢又犹豫的长 soak。
2. 自然处理豆，前 `45-60` 秒确实应该比水洗更保守。
3. 你的机器风控制弱，所以要用 `减载 + 排烟通道保持顺畅` 来替代“风量加大”。
4. 你要尽早开始收能量，而不是等一爆太近了才发现前段给多了。

对 `TANK200D` 的可执行翻译：

- 水洗浅烘：`150g-180g`，前段积极，黄点后开始提前收火
- 自然豆：起始火力比水洗低一点，或者更早做第一次降火
- 一爆前后：不要拖到爆后才大幅修锅，要在爆前就把下降趋势铺好

### 4. Uriel Oropeza：`2021` 墨西哥烘焙冠军，参加 `2023 World Roasting Championships`

这条案例的价值在于他公开写了参考起点：

- `270C` 预热
- 初始 `power 4`
- 初始 `airflow 2`
- `650g`

他给出的判断是：

- 如果豆子结构扛得住，更高的起始热量有机会带来更强的香气表现

### 这条案例怎么迁移到 `TANK200D`

不能把 `270C` 直接搬过来，因为：

- 机器不是一类
- 批量不是一类
- 传热结构也不是一类

但它仍然给你一个重要结论：

- 如果你要做比赛风格浅烘，不要默认“小机器就必须温吞”
- 对高密度、干净水洗豆，你可以测试一个更积极的入豆设定
- 代价是必须密切盯 `tipping / scorching / 烟气污染`

## C级资料：世界赛和国家赛里，小样机真正怎么被拿来赢比赛

### 6. Simo Kristidhi：`2019` 挪威烘焙冠军三连冠

这条案例非常值得你学，因为它和“比赛烘焙真正怎么准备”直接相关。

公开文里能确认：       

- `Simo` 是 `2019` 挪威烘焙冠军，而且是三连冠
- 比赛里他是唯一额外做了 `sample roast` 的选手
- 目的不是多烘一锅好玩，而是把样品机和比赛生产机 `Diedrich` 的试烘结果对照起来
- 他自己强调，比赛要赢，不只是会烘，而是要知道豆子潜力，以及生产线会怎么表现

### 这条案例对 `TANK200D` 的意义

这是我认为最该直接抄的方法：

1. 先用 `TANK200D` 做样品烘，找方向。
2. 再用更接近正式目标的装豆量，做“生产版本”。
3. 对齐的不是绝对温度，而是：
   - 脱水段长度
   - 黄点到一爆的推进节奏
   - 一爆后发展时间
   - 色值和失重

你的机器本质上就适合干这件事。

### 7. Norwegian Brewers Cup Champion：一爆后手动接管，是非常强的信号

这条 `ROEST` 案例里，最有价值的句子不是“赢了”，而是：

- 使用了 `Nordic Approach` 的一个哥伦比亚 profile
- `first crack` 以后转为 `manual control`
- 目的是更好控制最后的发展段

文中还保留了一条机器参数串，我把它整理成更易读的形式：

```text
ROEST 机器参数节点（机内设定，不是 TANK200D 豆温）
0:00  temp 200 / fan 50 / rpm 55
0:02  fan 40 / rpm 20
0:15  temp 150
1:00  temp 150 / fan 75 / rpm 65
1:30  temp 168
1:45  temp 175
5:05  temp 233
5:38  temp 237
8:00  temp 250
一爆后：转手动控制
```

这条线不能直接搬，但它说明了一件事：

- 比赛杯测分真正容易被拉开的，往往就在一爆后那一小段

对 `TANK200D` 的可执行翻译：

- 你至少要预设 `一爆后 2-3 个动作`
- 不要只写“听到爆就小降一点”
- 要提前规定：
  - 爆前最后一次降火在哪里
  - 爆后第几秒或第几个温区再降一次
  - 下豆条件是时间优先、温度优先，还是色值优先

### 8. Adrian Berg：怎么把样品线做成世界赛 Espresso 线

这条案例几乎是“比赛浓缩烘焙”的小样机教科书。

公开内容能确认：

- `Adrian Berg` 是 `2019`、`2020` 挪威咖啡师冠军
- 参加 `2021 WBC`
- 他用 `ROEST` 给自己的比赛浓缩做样品开发
- 他最后把烘焙时间拉到 `8-9` 分钟
- 发展时间做到 `90` 秒，`DTR` 约 `16%`

他的关键思路是：

- 高起点
- 短暂 soak
- 再次把热量拉回去
- 后段逐步下调
- 用稳定下降的 `RoR` 把时间拉长，而不是靠后段硬拖

### 对 `TANK200D` 的意义

如果你以后想用这台机练“比赛浓缩线”，这条案例非常实用：

- 很多小机用户会把浓缩烘得太短、太尖
- Adrian 这条线提醒你：真正服务比赛浓缩时，拉长总时长和发展时间，可能反而更合理
- 你的机器没有 ROEST 那么细的自动控制，所以更需要提前把 `黄点 / 一爆 / 下豆` 的三个区段写成计划表

### 9. 2023 世界赛样品机备赛案例集合

`Sample roasting for the World Stage` 这篇文章把一组世界赛选手放在一起看：

3. 再观察 `1-3 周` 的老化表现。
4. 最后决定正式比赛使用的曲线。

这和你用 `TANK200D` 的现实非常匹配，因为你的机器最擅长的正是：

- 少量多批
- 快速迭代
- 低成本试错

这对你是好消息。

- 样品开发
- 方向验证

## 我建议你现在就能开始练的 5 个动作

1. 把常规装豆量压到 `150g-180g`，专门练比赛风格浅烘。
2. 每支豆至少做一组“水洗版逻辑”和“自然版逻辑”对照，重点看前 `45-60` 秒是否需要更柔和。
3. 每锅都单独记录：
   - 黄点时间
   - 一爆开始时间
   - 一爆后发展时间
   - 失重
   - 色值
4. 把一爆后操作写成预案，而不是临场决定。
5. 做“样品锅 vs 正式锅”配对训练，目标是让两锅的风味方向一致，而不是让曲线截图长得一样。
