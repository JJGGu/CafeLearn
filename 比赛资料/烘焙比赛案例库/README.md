# 烘焙比赛案例库

这套目录把我这次从公开网络上整理到的咖啡烘焙比赛资料集中到一个地方，目标不是只堆链接，而是把“名次、原始资料、可用结论、曲线与操作逻辑”串起来。

## 目录结构

- `sources/raw_pdf/`
  - 官方历年 `WCRC` 排名 PDF
  - `2026` 官方规则、Roast Plan、总分表
- `sources/raw_html/`
  - 官方选手页、冠军公告、品牌采访页原文 HTML
- `sources/raw_html_machine/`
  - `TANK200/TANK200D` 相关实操页、同品牌赛事页、同类小型电热比赛机案例
- `sources/raw_img/`
  - 公开曲线图原图
- `sources/raw_pdf_machine/`
  - 机器相关补充 PDF，例如 `2025 WCRC` 规则
- `sources/外部链接补充.md`
  - 一些可读但不太稳定、或站点限制较强的补充页面
- `sources/机器相关补充链接.md`
  - 这台机器相关、以及同类比赛机的来源索引
- `analysis/`
  - 我写的中文整理、翻译、对曲线与操作的分析

## 推荐阅读顺序

1. `analysis/00_如何看比赛曲线与操作.md`
2. `analysis/01_2025_WCRC_前四名档案与解读.md`
3. `analysis/02_2024_WCRC_前四名档案与解读.md`
4. `analysis/03_2023_WCRC_前三名与两篇技术访谈.md`
5. `analysis/04_2022_中国区冠军马振星_曲线与操作解析.md`
6. `analysis/05_TANK200D_同机型与同类机型比赛案例.md`

## 这套资料能直接帮你看的东西

- 哪些年份的前几名名单已经拿到官方原件
- 哪些冠军/前三有公开采访，能看到操作思路
- 哪些资料和你现在的 `TANK200D` 更接近
- 哪些内容是“明确写出来的事实”
- 哪些内容是“根据曲线和规则做的推断”

## 使用提醒

1. 比赛公开资料里，最难拿到的往往不是“名次”，而是“完整生产曲线 + 具体按钮操作 + 杯测结果联动”。所以这套资料里会同时存在三种材料：
   - 官方排名/规则
   - 选手自述/访谈
   - 我基于曲线图和规则做的推断
2. 只要是我推断出来的，我都会明确写“推断”。
3. 公开网页有时会出现名字拼写不一致，我会优先以官方排名 PDF 或完整排名页为准。
