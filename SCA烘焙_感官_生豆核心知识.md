# SCA 烘焙、感官、生豆核心知识

这份文档不是课程宣传，而是给你作为烘焙师真正需要掌握的知识骨架。  
我会按 `SCA` 体系最相关的三大模块来写：

1. `Green Coffee`
2. `Roasting`
3. `Sensory Skills`

你把这三块学透，才有资格冲更高阶的烘焙水平。

---

## 一、Green Coffee：烘焙之前的判断能力

职业烘焙师和普通烘焙爱好者的一个巨大差别是：

**前者在投豆之前，就已经开始烘焙了。**

因为生豆信息本身就决定了烘焙策略。

## 1. 你必须掌握的生豆基础维度

### 1. 品种与种属

至少先把这些关系搞清：

1. `Arabica` 与 `Canephora/Robusta` 不是一个风味逻辑。
2. 不同品种对密度、酸质、花果香、甜感与结构的表现不同。
3. 同品种在不同产区、处理法、成熟度和储存状态下会有巨大差异。

对烘焙师的意义：

1. 高海拔高密度阿拉比卡通常更需要关注前段穿透和中段展开。
2. 低密度豆更容易表面先走。
3. 处理法和品种交互会改变你对“干净”“甜感”“发酵感”的目标设定。

### 2. 海拔、气候与成熟速度

你不必神化海拔，但必须理解：

1. 海拔常常和成熟速度、密度、酸质潜力有关。
2. 昼夜温差、降雨和采摘成熟度会影响风味成熟度。
3. 同一海拔不等于同一品质。

对烘焙师的意义：

1. 不要看到高海拔就无脑延长发展。
2. 不要看到低海拔就无脑加深。
3. 要结合密度、处理法和杯测信息综合判断。

### 3. 处理法

你至少要能系统说清楚：

1. 水洗。
2. 日晒。
3. 蜜处理。
4. 厌氧、酵母、二氧化碳浸渍等现代实验处理。

不是只会背定义，而是要知道对烘焙的影响：

1. 风味清晰度。
2. 发酵感风险。
3. 糖分和熟果表达。
4. 均匀度与耐烘性。

### 4. 含水率、密度、豆目、均匀度

这是烘焙操作最直接相关的物理信息。

你要知道：

1. 含水率高低会影响热吸收与前段表现。
2. 密度高的豆一般更吃能量，但不是无限吃。
3. 豆目不均匀时，焙度和发展均匀性更难控制。
4. 外观均匀不代表内部状态完全一致。

### 5. 缺陷与瑕疵

作为烘焙师，你不能只在熟豆阶段谈缺陷。  
生豆缺陷本身就会进入杯中。

你至少要建立三层理解：

1. 物理缺陷是什么。
2. 它可能怎样影响烘焙行为。
3. 它可能如何反映到杯中。

例如：

1. 黑豆、酸豆、虫蛀、破碎、贝壳豆、未成熟豆等。
2. 有些缺陷更容易造成局部过火或熟度不均。
3. 有些缺陷即使不显眼，也会让后段发木、脏、纸板、发酵异常。

### 6. 储存与年龄

生豆不是静止物。  
你必须理解：

1. 储存温湿度会影响稳定性。
2. 豆龄会影响香气活性与“陈感”。
3. 包装形式会影响老化速度。

对烘焙师的意义：

1. 同一条曲线，面对新季豆与陈季豆，结果可能完全不同。
2. 有时你以为是烘坏了，其实是原料状态变了。

## 二、生豆知识如何指导烘焙判断

你拿到新豆时，至少先回答下面 8 个问题：

1. 种属和品种是什么？
2. 处理法是什么？
3. 海拔和产区气候大概怎样？
4. 密度和含水率大概怎样？
5. 豆目是否均匀？
6. 外观是否干净？
7. 这支豆的目标风味应该是什么？
8. 这支豆最大的烘焙风险是什么？

这 8 个问题答不出来，你的第一炉设计基本只能靠猜。

## 三、Roasting：真正决定上限的工艺知识

## 1. 烘焙不是只看温度和总时间

你必须把烘焙理解成“热量管理下的物理与化学变化过程”。  
至少要掌握这几个核心概念：

1. 热传递。
2. 阶段变化。
3. 曲线控制。
4. 发展与焙度的关系。
5. 风味目标和生产一致性的平衡。

## 2. 三种热传递方式

### 传导

豆子接触滚筒表面、豆与豆之间接触时发生。

### 对流

热空气带来的热交换。

### 辐射

来自火源与高温表面的热辐射。

你的设备是小型辐射式直热滚筒机，所以你要特别意识到：

1. 辐射和直火特征更明显。
2. 局部过热风险更高。
3. 风门影响相对有限时，火力和载量更关键。
4. 表面反应和内部发展失衡的风险更大。

## 3. 烘焙过程中豆子发生了什么

你至少要理解这些变化：

1. 水分蒸发。
2. 颜色从绿到黄到棕的转变。
3. 香气前体逐步生成和转化。
4. 美拉德反应推进甜感和烘焙风味生成。
5. 一爆前后结构变化更剧烈。
6. 挥发性香气与结构发展常常相互拉扯。

错误理解是：

1. 只追求“把它烘熟”。

正确理解是：

1. 在不同阶段用不同能量策略，把风味潜力引导出来。

## 4. 四个关键阶段

### 投豆到回温

你要观察：

1. 机器被豆子拖低的程度。
2. 回温速度是否稳定。
3. 初段是否进入正常节奏。

### 回温到转黄

这是建立熟度基础的前段。  
你要关注：

1. 进热是否足够。
2. 是否过猛。
3. 是否给后面留出空间。

### 转黄到一爆

这是很多风味展开的关键阶段。  
你要关注：

1. 香气从生青转向甜香的速度。
2. 中段是否塌陷。
3. 是否只是表面上色，而不是内部推进。

### 一爆到下豆

你要关注：

1. 一爆开始是否清晰。
2. 发展期是否连贯。
3. 是否因为过度保守而停滞。
4. 是否因为拖太长而让风味钝化。

## 5. 什么叫发展，不是什么叫发展

很多人把“发展”简单理解成“一爆后时间”。  
这不够。

你必须同时看：

1. 一爆后时间。
2. 一爆前的热量状态。
3. 焙度终点。
4. 豆表与内部一致性。

同样是 90 秒发展期：

1. 前段强、中段稳、爆前平顺的样品。
2. 前段弱、中段塌、爆后硬拖的样品。

杯中会完全不同。

## 四、Roasting 模块真正该学的知识点

## 1. 设备与探针理解

你要懂：

1. 你的温度探针测到的是什么。
2. 探针滞后会带来什么误导。
3. 不同机器温度不可直接横比。
4. 风量、火力、滚筒转速在你的机器上哪个最有支配力。

对于你的机器，优先级大致是：

1. 火力。
2. 投豆量。
3. 预热一致性。
4. 下豆点。
5. 风门微调。

## 2. 曲线不是图好看，而是过程语言

你看曲线时，不要只看是否“漂亮”，而要回答：

1. 前段是否过冲？
2. 中段是否平顺？
3. 爆前有没有失控？
4. 发展期有没有停滞或拖尾？

## 3. 颜色与焙度

你要知道：

1. 焙度不是越浅越高级。
2. 焙度不是只看下豆温。
3. 颜色一致性与风味一致性高度相关，但不完全相同。
4. 标准杯测烘焙有其用途，不等于日常零售烘焙目标。

## 4. 失重率

失重率不是万能指标，但很有用。  
它可以帮助你：

1. 比较同豆不同烘法的深浅差异。
2. 检查生产重复性。
3. 发现异常批次。

但你不能只靠失重率判断好坏。  
同样失重率，也可能对应完全不同的风味质量。

## 5. 烘焙缺陷识别

你必须能识别至少这些典型问题：

1. 欠发展。
2. 过发展。
3. 烘焙过快。
4. 烘焙过慢。
5. 烘焙不均。
6. 烟熏污染。
7. 表面受热过强导致的焦尖感。

并且要会把它们映射回过程：

1. 哪个阶段出问题。
2. 是热量太多、太少，还是时机不对。

## 6. 样品烘焙与生产烘焙不是一回事

你以后必须学会区分：

### 样品烘焙

目标：

1. 快速获得信息。
2. 尽量公平地暴露生豆特征。

### 生产烘焙

目标：

1. 为特定消费场景优化风味。
2. 追求一致性、效率与可复制。

很多初学者把这两件事混在一起，所以会：

1. 用生产逻辑去做样品判断。
2. 用样品逻辑去设计零售烘焙。

## 五、Sensory Skills：你真正的“控制台”

## 1. 感官不是附属能力，是主系统

烘焙师的感官不是烘完才用。  
它贯穿：

1. 生豆判断。
2. 烘焙时观察。
3. 烘后评估。
4. 杯测与修正。

## 2. 你必须掌握的感官科学基础

### 味觉

至少知道：

1. 甜、酸、苦、咸是基础味觉。
2. 咖啡中的“酸”是风味维度，不等于缺陷。
3. 涩不是基础味觉，而是口腔收敛感。

### 嗅觉

至少知道：

1. 干香、湿香、啜吸时的回香不是同一层次。
2. 很多“风味词”本质上是嗅觉而不是味觉。

### 口感

至少知道：

1. 厚薄。
2. 滑感。
3. 干燥感。
4. 金属感。
5. 粗糙感。

这些都必须学会分开说。

## 3. 描述风味时不要只报水果名

世界级烘焙师在杯测时不会只说：

1. 蓝莓。
2. 柠檬。
3. 花香。

他们还会说：

1. 清晰度如何。
2. 强度如何。
3. 与甜感是否融合。
4. 结构是否完整。
5. 后段是否拉得开。

这才是真正影响烘焙修正的部分。

## 4. 负面问题的职业化识别

你必须能稳定说清：

1. 生青感。
2. 木质感。
3. 纸板感。
4. 发闷。
5. 烟感。
6. 灰感。
7. 发酵过度感。
8. 苦涩分离。
9. 酸尖不融。
10. 尾段干燥。

如果这些说不清，你每次修正都会像蒙眼开车。

## 六、把三大模块连起来

## 1. 生豆决定风险边界

例如：

1. 高密度水洗豆，前段不足时更容易出现“熟度不够但表面香气有”的假象。
2. 日晒豆如果中后段处理失衡，更容易出现闷甜或发酵感脏化。

## 2. 烘焙决定表达方式

同一支豆可以被表达成：

1. 更花果清晰。
2. 更甜更厚。
3. 更平衡但不那么张扬。

## 3. 感官决定你能否识别差异

如果你不能稳定识别这些表达差异，你就无法真正“设计风味”。

## 七、SCA 课程层级对你意味着什么

你不一定非要立刻去报所有课，但你要用课程层级来校验自己的能力。

## Foundation 水平，你至少要能做到

1. 说清基础概念。
2. 正确操作基础流程。
3. 识别明显差异和典型问题。

## Intermediate 水平，你至少要能做到

1. 控制变量。
2. 解释因果。
3. 做重复性更高的评估。

## Professional 水平，你至少要能做到

1. 设计系统。
2. 训练他人。
3. 在真实业务中做决策。

如果你想成为顶尖烘焙师，你最终必须达到的是：

**不仅自己会做，还能把体系讲清、复制、管理和优化。**

## 八、你作为烘焙师必须掌握的知识清单

下面这份清单可以作为你未来 6-12 个月的知识检查表。

### Green Coffee 清单

1. 品种、种属与处理法的基础差异。
2. 海拔、密度、含水率、豆目与均匀度。
3. 常见物理缺陷及其潜在杯中表现。
4. 生豆储存与老化逻辑。
5. 样品评估与采购基础。

### Roasting 清单

1. 热传递三方式。
2. 设备探针与数据局限。
3. 四阶段过程判断。
4. 焙度、时间、发展、失重率之间的关系。
5. 常见烘焙缺陷与修正逻辑。
6. 样品烘焙与生产烘焙差异。

### Sensory 清单

1. 基础味觉与嗅觉区别。
2. 结构、清晰度、平衡、尾段的定义。
3. 风味词与参考物训练。
4. 三角杯、盲测、校准。
5. 负面风味识别与稳定复述。

## 九、你现在最该补的知识短板

结合你前面的描述，你当前最可能的短板排序是：

1. 感官语言和结构判断。
2. 杯中问题到过程问题的映射。
3. 生豆信息对烘焙策略的前置判断。
4. 单变量实验背后的理论解释。

也就是说，你不是没有烘焙经验，而是：

**经验很多，但还没有被整理成系统知识。**

这正是 `SCA` 体系最适合补你的地方。
