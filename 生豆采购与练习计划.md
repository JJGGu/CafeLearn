## 二、24 周分阶段豆单与采购计划

### 核心原则

1. **前 8 周只用 1-2 支豆**，建立基准，不要频繁换豆。
2. **第 9-16 周引入对比豆**，按产区、处理法、密度做对比实验。
3. **第 17-24 周扩展豆单**，练习策略切换和快速判断。
4. 每支豆至少买 **1kg 生豆**（约 5 炉），确保有足够炉数做重复和对比。

## 四、生豆到货后的处理

### 到货检查清单

1. ☐ 核对品种、产区、处理法是否与购买信息一致
2. ☐ 打开包装闻干香：应有清新的青草/谷物香，不应有霉味、陈味
3. ☐ 取 50g 样品平铺观察：豆目大小是否均匀，有无明显瑕疵豆
4. ☐ 拍照记录外观
5. ☐ 记录到货日期
6. ☐ 如有条件，测含水率

### 瑕疵豆分拣

训练期间建议每次烘焙前做简单手选：

1. 挑出明显的黑豆、虫蛀豆、破碎豆
2. 挑出明显的未成熟豆（颜色偏浅、偏小）
3. 挑出贝壳豆
4. 不需要追求完美，但要建立"看豆"的习惯

## 五、生豆存放

### 存放条件

| 项目 | 要求 |
|------|------|
| 温度 | 阴凉处，避免高温（< 25°C） |
| 湿度 | 干燥，避免潮湿 |
| 光照 | 避光 |
| 容器 | 原包装密封袋 或 密封罐/密封袋 |
| 标记 | 每袋贴标签（见标签模板） |

### 存放位置建议

- 家里阴凉干燥的柜子即可
- 不要放冰箱（容易受潮）
- 不要放阳台或窗边
- 开封后尽量排气密封
