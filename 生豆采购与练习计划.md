## 二、采购策略先看 5 条原则

### 1. 前 6 周最重要的是“同 lot、够量、便宜、干净、豆型相对集中”

高频训练最大的隐性问题不是价格，而是**同一支基准豆不够用**。  
如果你前 `6` 周都想拿同一支豆建立载量、阶段和复现逻辑，那就不要只买 `1-2kg`。

更实际的思路是：

1. 基准豆优先选传统处理、水洗、信息完整、价格不过高的豆。
2. 理想状态下，前 `6` 周的主基准豆要尽量锁同一 lot。
3. 优先选**豆径比较集中、圆豆/平豆不混乱**的豆当基准豆。
4. 如果拿不到足够量，就主动把计划拆成“主基准豆 + 次基准豆”，而不是中途被动断豆。

### 2. 高价竞标豆和重发酵豆不要拿来做高频主训练豆

像竞标批次、重发酵、实验处理豆可以做上限校准，但不适合承担高频重复训练。  
原因很简单：

1. 单炉成本高。
2. 杯感变量多。
3. 很难把问题明确归因到烘焙阶段。
4. 同 lot 数量往往不够大。

### 3. 你需要两种采购角色，不是一种

一套高频训练里，豆子至少分成两类：

1. **基础训练豆**：便宜、干净、信息完整、量大，用来建立载量和阶段控制。
2. **策略切换豆**：高密度水洗、日晒/蜜处理、低密度豆，用来练切换能力。

### 4. 不要把 200g 载量写进采购逻辑里当默认值

采购时你要按“训练目标”选豆，而不是按“固定 190g / 200g”选豆。

更合理的理解是：

1. 高密度浅烘训练豆，要允许你去测 `120g / 150g / 180g`。
2. 通用工作豆，要允许你去测 `150g / 180-190g`。
3. 复现豆和中浅到中度豆，才更适合 `190-200g`。

### 5. 基准豆优先买“物理变量少”的，策略豆再故意买“物理变量不同”的

这条很重要。  
如果你一开始就买一支密度高、处理法复杂、豆径还很散的豆，那你之后很难判断到底是曲线问题还是豆子本身噪音。

更合理的采购顺序是：

1. 前 `1-6` 周：优先买 **豆径相对集中、豆型相对单纯** 的豆。
2. 第 `7-9` 周：再故意引入 **更大粒、圆豆、低密度、小粒豆** 这些物理差异。
3. 第 `10-12` 周：才让“复杂处理法 + 复杂豆型”同时进场。

### 第 1 阶段选豆时，多加一个物理筛选标准

只要有选择空间，优先选：

1. 豆径集中、混粒感不强的批次。
2. 圆豆比例低，或至少圆豆信息明确、可单独处理的批次。
3. 同 lot 数量足够，避免一边训练一边换成不同筛网结构的新 lot。

1. 杯感能清晰反映过程变化。
2. 成本不能太高。
3. 量要够。

水洗豆在这三点上都更适合作为高频映射豆。

## 七、购买时必须确认的信息

每支豆买之前，至少确认下面这些信息：

1. 产区。
2. 品种。
3. 处理法。
4. 海拔。
5. 采收年份。
6. 含水率（如有）。
7. 密度（如有）。
8. 水活度（如有）。
9. 筛网规格 / 豆径信息（如有）。
10. 豆型信息：平豆、圆豆、长豆，或圆豆比例。
11. 均匀度信息：是否明显混粒、是否同 lot 稳定。
12. 杯测信息（如有）。
13. 是否能提供同 lot 的持续补货。

> **对于这版高频训练，同 lot 是否能持续补货，几乎和价格一样重要。**

## 八、生豆到货后的处理

### 到货检查清单

1. ☐ 核对品种、产区、处理法是否与购买信息一致
2. ☐ 闻干香：应清新，不应有霉味、陈味
3. ☐ 取少量样品平铺观察：豆目是否均匀，有无明显瑕疵豆、混粒、混豆型
4. ☐ 随机抽 `100` 粒，估计主尺寸组占比；有筛网就记筛号，没有筛网就至少做“小 / 中 / 大”分档
5. ☐ 记录到货日期、供应商、lot 信息、开袋日期
6. ☐ 分装并贴标签，避免后期混 lot

### 分装建议

1. 高频主训练豆按 `1kg` 或 `500g` 分装。
2. 同一支豆的不同 lot 必须单独标记。
3. 如果一支豆明显混粒，可以额外留一小袋“主粒组样本”或“圆豆组样本”用于验证豆型影响。
4. 挑战豆和标杆豆可以按 `500g` 或更小规格分装。

### 存放建议

1. 避光。
2. 室温阴凉。
3. 防潮。
4. 不要频繁开封同一大袋。

## 九、这一版采购计划最终要服务什么

这份采购计划不是为了“囤很多好豆”。  
它真正要服务的是：

1. 让你前 `6` 周有足够稳定的同 lot 基准豆。
2. 让你第 `7-9` 周能切到明确不同类型的策略豆。
3. 让你第 `10-12` 周同时有复现豆和挑战豆。
4. 让你的高频训练不会因为断豆、换 lot 或单炉成本过高而中断。

如果这 4 件事能满足，这份采购计划就是有效的。
