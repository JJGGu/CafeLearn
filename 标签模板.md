# 烘焙样品标签模板

## 使用说明

- 建议打印在 **不干胶标签纸** 上（推荐尺寸：70mm × 40mm 或 50mm × 30mm）
- 也可以手写在空白标签贴纸上
- 每袋熟豆必须贴标签，这是建立系统记录的基础

---

## 标签内容（必填项）

```
┌─────────────────────────────────┐
│  编号：________                  │
│  豆名：________                  │
│  产区：________  处理法：________ │
│  烘焙日期：____年____月____日     │
│  批次重量：____g → ____g         │
│  失重率：____%                   │
│  目标/变量：____________________ │
│  最早杯测：____月____日          │
│  备注：________________________ │
└─────────────────────────────────┘
```

---

## 各字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| **编号** | 格式：年月日-序号，如 240315-01 表示 3月15日第1炉 | 260310-02 |
| **豆名** | 生豆名称，简写即可 | 耶加雪菲 科契尔 |
| **产区** | 国家 + 产区 | 埃塞 耶加 |
| **处理法** | 水洗/日晒/蜜处理等 | 水洗 |
| **烘焙日期** | 下豆日期 | 2026.03.10 |
| **批次重量** | 生豆重量 → 熟豆重量 | 300g → 252g |
| **失重率** | (生豆-熟豆)/生豆 × 100% | 16% |
| **目标/变量** | 本炉唯一目标或唯一改动变量 | 发展 90s / 基准复现 |
| **最早杯测** | 烘焙日期 + 养豆天数 | 3月15日 |
| **备注** | 任何补充信息 | 一爆密集 / 颜色偏浅 |

---

## 颜色编码建议（可选）

用不同颜色的标签贴纸区分用途：

| 颜色 | 用途 |
|------|------|
| **白色** | 常规训练样品 |
| **黄色** | 基准/参考样品（勿动） |
| **红色** | 刻意做的"错误样品"（用于对比训练） |
| **绿色** | 最佳样品（留作参考标准） |

## 打印建议

1. **手写方案**：购买空白不干胶标签（淘宝搜"空白不干胶标签贴纸"），手写填写
2. **打印方案**：用 A4 不干胶标签纸，在 Word 中排版后打印
3. **热敏打印方案**：如果有标签打印机（如精臣），可以设计模板批量打印

> 新手阶段推荐手写，因为写的过程本身就是强化记忆。
