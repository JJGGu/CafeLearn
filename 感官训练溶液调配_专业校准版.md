# 感官训练溶液调配专业校准版（100mL 棕瓶适配）

> 版本：`2026-03-25`  
> 目标：给你一套**尽量靠近公开可核实专业来源**、并且适合你现在 `100mL` 棕瓶、`0.01g` 电子秤和现有粉剂条件的训练液方案。  
> 这份文件不再把你之前那份本地采购文档当成唯一依据，而是优先依照公开能查到的专业来源重建。

---

## 一、先说结论

### 1. 我确认到的公开专业依据

1. `SCA` 官网明确写到，`Coffee Taster's Flavor Wheel` 的基础是 `World Coffee Research Sensory Lexicon`。
2. `WCR Sensory Lexicon 2.0` 公开给出了咖啡领域最关键的一批参考液锚点：
   - 甜：`1.0%` 蔗糖
   - 酸：`0.015%`、`0.05%` 柠檬酸
   - 苦：`0.01%`、`0.02%`、`0.035%`、`0.05%` 咖啡因
   - 咸：`0.15%` 氯化钠
   - 干燥：`0.05%`、`0.07%` 明矾
   - 金属：`0.10%` 氯化钾
   - 厚度：`5%` 蔗糖
   - 柠檬酸属性：`0.025%`、`0.05%`
   - 苹果酸属性：`0.05%`、`0.10%`
3. `WCR` 没有公开给出所有味觉都完整的“低 / 中 / 高 / 很高”阶梯，也没有公开给出 `酒石酸` 的咖啡专用参考液梯度。
4. 为了补足“甜、咸、酸强度阶梯”和 `酒石酸`，我补用了**公开的专业感官训练文献**：
   - `MDPI 2024` 的葡萄酒评审员训练方法论文，里面公开列了水溶液里的味觉识别和强度排序浓度。
   - `FAO` 的感官训练指南，里面总结了 `Meilgaard`、`ASTM` 等来源常用的训练浓度范围。

### 2. 这意味着什么

这次我会把配方分成两层：

1. **严格核实锚点**：我只放公开来源里确实写出来的浓度。
2. **训练阶梯建议**：在严格锚点基础上，用公开专业训练表把低中高补齐。

这样你可以先用“严格核实锚点”开练；  
如果想做强度排序，再上“训练阶梯”。

## 二、来源边界和我不想骗你的地方

### 1. 没有一个“全世界唯一公开统一”的咖啡训练液配方表

这是事实。

公开可核实的咖啡领域最强来源是 `WCR Sensory Lexicon`，  
但它更像是**锚点体系**，不是一整套把所有味觉都拉成四档的家庭训练手册。

### 2. `SCA` 的公开网页没有把完整课堂配液表免费放出来

我核到了 `SCA` 官网说明：它的风味轮是建立在 `WCR Lexicon` 之上的。  
但我**没有核到**一份公开的、逐项写死浓度的 `SCA` 课堂配液表。

所以这次最稳妥的做法是：

1. 咖啡相关的基础锚点，优先用 `WCR`
2. 甜、咸、酸排序梯度，补用公开的专业感官训练论文
3. 明确标出哪一项是“咖啡公开锚点”，哪一项是“专业训练扩展”

### 3. “不同酸做成同一克数”不等于“标准”

这点非常重要。

`柠檬酸`、`苹果酸`、`酒石酸` 的感知强度和感知形状都不同。  
你不能简单地把三种酸都配成一样的 `0.05%`，然后默认它们就是“同强度标准液”。

更稳妥的训练顺序是：

1. 先练每种酸自己的公开锚点
2. 再练各自的强度阶梯
3. 最后再做酸型横向比较

## 五、三种酸怎么练，才更接近“标准”

### 1. 柠檬酸

如果你要的是**标准感官锚点**，优先用：

1. `0.015%`
2. `0.025%`
3. `0.05%`

这三档都能从 `WCR` 的公开系统里找到依据。

### 2. 苹果酸

如果你要的是**标准感官锚点**，优先用：

1. `0.05%`
2. `0.10%`

这两档是我能公开核到的 `WCR` 苹果酸锚点。  
如果你坚持要第三档，我建议你先别急着补，因为那一步已经不是“公开核实锚点”了。

### 3. 酒石酸

酒石酸没有在 `WCR` 公开文档里给出咖啡专用浓度锚点。  
但在公开的专业评审员训练文献里，水溶液强度梯度是有的。

对你现在的器材，我建议把酒石酸按下面这组做：

1. `0.025%`
2. `0.05%`
3. `0.075%`
4. `0.10%`

## 六、母液法：你现在这套器材必须学会这个

### 1. 为什么要母液法

即使你有 `0.01g` 电子秤，下面这些数也不适合直接硬称：

1. `0.015g`
2. `0.025g`
3. `0.035g`
4. `0.075g`

原因不是你不会操作，而是称本身的分辨率已经太接近目标值。

### 2. 最实用的方法：做 `1.0%` 母液

公式：

1. 取目标粉剂 `1.00g`
2. 加水到 `100mL`
3. 这就是 `1.0%` 母液

你可以分别做：

1. 柠檬酸母液
2. 苹果酸母液
3. 酒石酸母液
4. 咖啡因母液

### 3. 用 `1.0%` 母液配 `100mL` 成品液

| 目标浓度 | 取 `1.0%` 母液 | 再加水到 |
|---------|----------------|---------|
| `0.010%` | `1.0mL` | `100mL` |
| `0.015%` | `1.5mL` | `100mL` |
| `0.025%` | `2.5mL` | `100mL` |
| `0.035%` | `3.5mL` | `100mL` |
| `0.050%` | `5.0mL` | `100mL` |
| `0.075%` | `7.5mL` | `100mL` |
| `0.100%` | `10.0mL` | `100mL` |

如果你没有 `1mL` 级量取工具，建议补一支 `10mL` 注射器或刻度移液管。

## 八、哪些我不建议你现在做

### 1. 不建议做“碱性标准液”

这版我没有给你加碱性标准液，比如小苏打、碳酸钠之类。  
原因不是不能喝，而是：

1. 它不是咖啡基础味觉校准里的常规公开锚点
2. 很容易把训练从“感官标准化”带偏到“家庭化学实验”
3. 在家庭环境里，碱带来的口感和后味也更容易让你误判

### 2. 不建议一开始就做“等酸强比较”

公开研究已经说明：  
同一种酸在水里和在其他基质里，等强度点并不完全一样；不同酸之间也会因为体系不同而变化。

所以第一阶段先不要追求：

1. 柠檬酸 `0.05%`
2. 苹果酸 `x%`
3. 酒石酸 `y%`

三杯一定要主观上“完全一样酸”。

先把各自锚点练稳，反而更标准。

## 九、这次方案怎么用

如果你只想一句话结论：

1. **基础味觉**：用 `WCR` 锚点
2. **甜咸强度**：用公开专业训练梯度补成 `低 / 中 / 高`
3. **酸型**：柠檬酸和苹果酸用 `WCR` 锚点，酒石酸用公开专业训练水溶液梯度
4. **低克重项目**：全部走母液法

这样做的好处是：

1. 你喝到的不是随便猜的比例
2. 你知道哪些是“公开核实”
3. 你知道哪些是“专业训练扩展”

这比把所有东西都写成一个看起来很整齐的梯度表，其实更标准。

## 十、如果你觉得 1% 糖太淡，该怎么处理

这是一个很好的提醒，而且你的感觉是正常的。

### 1. `1.0%` 蔗糖本来就不是“很明显的甜”

`WCR Sensory Lexicon` 里，`1.0% sucrose solution` 的甜味强度只标到 `1.0`。  
这说明它本来就是一个**偏轻、偏校准用**的锚点，不是“新手一喝就会觉得很明显”的训练液。

同样地，`WCR` 里的低档酸、低档苦，也不是都设计成“特别好认”。  
有些就是为了校准轻度刺激。

### 2. 所以训练要分成两套，不要混成一套

更实用的做法是：

1. **放大识别组**：先把味觉类别练清楚
2. **标准校准组**：再回到 `WCR` 锚点

你之前用 `1g / 3g / 6g` 糖去配，如果是按 `100mL` 算，那其实就是：

1. `1%`
2. `3%`
3. `6%`

这组对于“先把甜感练出来”其实是有价值的。  
问题不在于它不标准，而在于它更像**放大识别组**，不是咖啡专业里用于轻度校准的锚点组。

### 3. 我建议你现在这样分

#### 甜感

1. 放大识别组：`1% / 2% / 4%`
2. 标准校准组：`0.5% / 1.0% / 1.5%`

#### 咸感

1. 放大识别组：`0.10% / 0.20% / 0.40%`
2. 标准校准组：`0.10% / 0.15% / 0.20%`

#### 柠檬酸

1. 放大识别组：`0.025% / 0.05% / 0.10%`
2. 标准校准组：`0.015% / 0.025% / 0.05%`

#### 苦味

1. 放大识别组：`0.02% / 0.035% / 0.05%`
2. 标准校准组：`0.01% / 0.02% / 0.035%`

这里的“放大识别组”不是我随便拍脑袋给的，  
而是参考了公开感官训练资料里常见的训练范围。

不过苦味我特意收得更保守。  
原因是家庭训练里不值得为了“更明显”把咖啡因推太高。

### 4. 给你的实际建议

如果你现在喝 `1%` 糖明显觉得淡，那不要硬逼自己从最淡的校准锚点开始。  
更合理的顺序是：

1. 先用放大识别组，把“甜就是甜、咸就是咸、酸就是酸、苦就是苦”练稳
2. 再逐步下探到标准校准组
3. 最后再做接近 `WCR` 的低强度锚点

这不叫“不标准”，这叫**分阶段训练**。

### 5. 这件事对其他味觉也同样成立

是的，其他味觉也会有这个问题，尤其是：

1. 低档柠檬酸
2. 低档咖啡因
3. 低档盐

这些在专业体系里本来就可能是“可感知但不夸张”的设置。  
如果你一开始就只练这类轻刺激，很容易出现：

1. 觉得自己尝不出来
2. 怀疑配错了
3. 训练挫败感很强

所以前期用“放大识别组”，后期回到“标准校准组”，反而更专业。

### 1. 我最推荐你先配的 `16` 瓶

这 `16` 瓶优先解决四件事：

1. 甜、咸、酸、苦能不能稳定认出来
2. 强度能不能排顺序
3. 柠檬酸、苹果酸、酒石酸能不能分类型
4. 训练时不要因为太淡而卡住

| 编号 | 内容 | `100mL` 配方 | 用途 |
|------|------|-------------|------|
| `SW-10` | 蔗糖 `1%` | 蔗糖 `1.00g` | 甜感校准锚点 |
| `SW-20` | 蔗糖 `2%` | 蔗糖 `2.00g` | 甜感放大识别 |
| `SW-40` | 蔗糖 `4%` | 蔗糖 `4.00g` | 甜感高档排序 |
| `SA-10` | 食盐 `0.10%` | 食盐 `0.10g` | 咸感低档 |
| `SA-15` | 食盐 `0.15%` | 食盐 `0.15g` | 咸感校准锚点 |
| `SA-20` | 食盐 `0.20%` | 食盐 `0.20g` | 咸感高档 |
| `CT-025` | 柠檬酸 `0.025%` | 建议母液法 | 柑橘酸中档 |
| `CT-050` | 柠檬酸 `0.05%` | 柠檬酸 `0.05g` | 柑橘酸锚点 |
| `CT-100` | 柠檬酸 `0.10%` | 柠檬酸 `0.10g` | 柑橘酸放大高档 |
| `BT-020` | 咖啡因 `0.02%` | 咖啡因 `0.02g` | 苦味中档 |
| `BT-035` | 咖啡因 `0.035%` | 建议母液法 | 苦味锚点高档 |
| `BT-050` | 咖啡因 `0.05%` | 咖啡因 `0.05g` | 苦味放大高档 |

### 2. 如果你还有余力，再补这 `3` 瓶

| 编号 | 内容 | `100mL` 配方 | 用途 |
|------|------|-------------|------|
| `AL-05` | 明矾 `0.05%` | 明矾 `0.05g` | 干燥感 |
| `KC-10` | 氯化钾 `0.10%` | 氯化钾 `0.10g` | 金属感 |
| `SW-50` | 蔗糖 `5%` | 蔗糖 `5.00g` | 厚度 / 黏稠感 |

### 3. 如果你想刚好配满 `20` 瓶

可以再加这 `1` 瓶：

| 编号 | 内容 | `100mL` 配方 | 用途 |
|------|------|-------------|------|
| `AL-07` | 明矾 `0.07%` | 明矾 `0.07g` | 高强度干燥感 |

### 4. 这几天的训练顺序

#### 第 1 天

1. `SW-10 / SW-20 / SW-40`
2. `SA-10 / SA-15 / SA-20`

#### 第 2 天

1. `CT-025 / CT-050 / CT-100`
2. `MA-050 / MA-100`
3. `TA-050 / TA-100`

#### 第 3 天

1. `BT-020 / BT-035 / BT-050`
2. 回头混入 `SW-20`、`SA-15`、`CT-050` 做盲测

#### 第 4 天

1. 如果已经补了扩展瓶：`AL-05 / KC-10 / SW-50`
2. 再做一次“甜 / 咸 / 柠檬酸 / 苹果酸 / 酒石酸 / 苦”混合盲测

### 5. 一个很实用的操作建议

如果你这周只想真的开始，而不是把自己困在调配里，  
那我建议你先配：

1. 上面的 `16` 瓶
2. 暂时先不配 `AL-07`
3. 如果当天精力一般，就连 `AL-05 / KC-10 / SW-50` 也可以第二批再补

这样你第一周就已经能把最核心的味觉框架搭起来了。

### 2. 建议母液法的两瓶

| 瓶号 | 内容 | 推荐做法 | 为什么 |
|------|------|---------|------|
| `CT-025` | 柠檬酸 `0.025%` | 用 `1.0%` 柠檬酸母液取 `2.5mL`，再加水到 `100mL` | `0.025g` 太接近秤分辨率 |
| `BT-035` | 咖啡因 `0.035%` | 用 `1.0%` 咖啡因母液取 `3.5mL`，再加水到 `100mL` | `0.035g` 不够稳 |

### 3. 最推荐的配制顺序

为了减少残留和串味，建议按这个顺序做：

1. 蔗糖三瓶：`SW-10`、`SW-20`、`SW-40`
2. 食盐三瓶：`SA-10`、`SA-15`、`SA-20`
3. 柠檬酸两瓶：`CT-050`、`CT-100`
4. 苹果酸两瓶：`MA-050`、`MA-100`
5. 酒石酸两瓶：`TA-050`、`TA-100`
6. 先做柠檬酸母液，再配 `CT-025`
7. 最后做咖啡因两瓶：`BT-020`、`BT-050`
8. 再做咖啡因母液，最后配 `BT-035`

### 4. 两瓶母液怎么做

#### 柠檬酸母液 `1.0%`

1. 取一个空瓶，贴 `CT-STOCK-1.0%`
2. 称柠檬酸 `1.00g`
3. 加水到 `100mL`
4. 摇匀

#### 咖啡因母液 `1.0%`

1. 取一个空瓶，贴 `BT-STOCK-1.0%`
2. 称咖啡因 `1.00g`
3. 加水到 `100mL`
4. 摇匀
5. 单独放，贴清楚“勿误饮”

### 5. 第一周最简单的练法

#### 第 1 轮：只练主味觉

1. `SW-20`
2. `SA-15`
3. `CT-050`
4. `BT-020`

#### 第 2 轮：开始排强度

1. `SW-10 / SW-20 / SW-40`
2. `SA-10 / SA-15 / SA-20`
3. `CT-025 / CT-050 / CT-100`
4. `BT-020 / BT-035 / BT-050`

#### 第 3 轮：开始分酸型

1. `CT-050`
2. `MA-050`
3. `TA-050`

然后再做：

1. `CT-100`
2. `MA-100`
3. `TA-100`

### 6. 你今天如果只想先开一半

那就先配这 `8` 瓶：

1. `SW-20`
2. `SW-40`
3. `SA-15`
4. `SA-20`
5. `CT-050`
6. `MA-050`
7. `TA-050`
8. `BT-020`

这 `8` 瓶已经足够你开始练“甜、咸、酸型、苦”的核心框架。
