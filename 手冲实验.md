# 手冲实验计划（按影响优先级版）

> 目标：把手冲里会明显影响风味的变量按“影响大小”和“研究顺序”排好，让你可以根据自己的时间，逐步建立一套从问题判断到参数修正的实验系统。  
> 这份文档仍然建议你先从高影响因素开始，但现在会把完整变量都列出来，方便你后续分阶段深入。

---

## 一、起步原则

- 先追求“可重复”，再追求“最好喝”。
- 同一轮实验只改 `1` 个变量。
- 同一轮实验最好用同一支豆子，并尽量在 `2-3` 天内完成。
- 同一支豆子尽量保持相近养豆天数。
- 固定滤杯、滤纸、磨豆机、水、手法和注水节奏。
- 总时长是观察指标，不是第一阶段的目标本身。
- 每个水平至少做 `2` 杯；如果时间有限，至少先做 `1` 杯找方向，再挑 `2` 个关键水平复做。

---

## 二、建议基准配方

| 项目 | 建议值 | 说明 |
|------|--------|------|
| 豆子 | 同一支豆 | 建议先用你已经熟悉、瑕疵少的豆子 |
| 粉量 | `15g` | 起步最容易比较 |
| 水量 | `240g` | 对应 `1:16` |
| 粉水比 | `1:16` | 先作为对照组 |
| 水温 | `92C` | 中浅烘常用起点 |
| 研磨度 | 你当前“正常好喝”的刻度 | 先当作基准点 |
| 闷蒸 | `30-40g / 30s` | 全程保持一致 |
| 注水方式 | `3` 段注水 | 全程保持一致 |
| 总时长 | `2:30-3:00` | 只作为 sanity check |

如果是中深烘，可以把整套水温下调 `2C` 左右，再做同样的实验。

---

## 三、先做重复性验证

在正式实验前，先用基准配方连续做 `3` 杯。  
满足下面两条，再进入变量实验：

- 总时长差异尽量不超过 `10-15` 秒。
- 风味方向基本一致，比如都偏酸薄、或都偏平衡，不要一杯酸一杯苦。

如果重复性都不稳定，先不要急着改参数，先把以下动作固定：

- 注水高度
- 注水速度
- 绕圈范围
- 断水时机

---

## 四、变量优先级总表

| 优先级 | 变量 | 影响等级 | 建议优先级 | 主要影响 |
|------|------|---------|-----------|---------|
| `1` | 研磨度 | 很高 | 最先研究 | 萃取效率、流速、苦涩与甜感平衡 |
| `2` | 粉水比 | 很高 | 最先研究 | 浓度、厚度、集中度、整体平衡 |
| `3` | 水温 | 很高 | 最先研究 | 萃取强度、香气展开、后段苦涩 |
| `4` | 注水结构 / 段数 | 高 | 第二阶段 | 接触时间分布、均匀性、层次感 |
| `5` | 扰动强度 | 高 | 第二阶段 | 萃取均匀性、甜感、杂味、涩感 |
| `6` | 总粉量 / 粉层厚度 | 高 | 第二阶段 | 流速、萃取效率、风味集中方式 |
| `7` | 注水速度 / 流量 | 中高 | 第二阶段 | 水位、接触时间、萃取推动力 |
| `8` | 闷蒸时间 / 闷蒸水量 | 中高 | 第二阶段 | 排气、前段均匀性、香气释放 |
| `9` | 滤杯 | 中高 | 第三阶段 | 流速模型、厚度、干净度、层次 |
| `10` | 滤纸 | 中 | 第三阶段 | 流速、油脂保留、干净度 |
| `11` | 水质 / 矿物质组成 | 很高 | 第三阶段但值得长期研究 | 甜感、酸质、清晰度、口感骨架 |
| `12` | 注水高度 / 水柱集中度 | 中 | 第三阶段 | 粉床扰动、萃取局部强度 |
| `13` | 断水时机 / 水位控制 | 中 | 第三阶段 | 通道风险、萃取均匀性 |

如果你的时间有限，只研究前 `5-6` 个变量，就已经足以大幅提升手冲稳定性。

---

## 五、第一阶段：最高影响因素

### 1. 实验 A：研磨度

这是第一优先级，因为它通常对萃取和风味变化最直接。

固定不变：

- 粉量 `15g`
- 水量 `240g`
- 粉水比 `1:16`
- 水温 `92C`
- 闷蒸 `30s`
- 注水方式不变

变量设置：

| 组别 | 设置 |
|------|------|
| A1 | 比基准粗一点 |
| A2 | 基准 |
| A3 | 比基准细一点 |

研磨调整建议：

- 如果磨豆机刻度较细，每次改 `2-3` 小格。
- 如果磨豆机刻度较粗，每次改 `1` 大格。

重点观察：

- 总时长是否明显变化
- 酸是否从尖、薄变得更圆润
- 甜感是否提升
- 后段是否开始发苦、发干、发涩

常见趋势：

| 方向 | 可能表现 |
|------|---------|
| 更粗 | 更快、偏淡、偏酸、结构松 |
| 更细 | 更慢、甜感上升、厚度上升，但过头会苦涩 |

这一轮的目标不是找“神配方”，而是找到你的豆子从“偏生”到“偏过”的区间。

### 2. 实验 B：粉水比

等研磨度找到一个相对顺手的点，再研究浓度与口感平衡。

固定不变：

- 研磨度使用实验 A 里最顺手的刻度
- 水温 `92C`
- 闷蒸与注水方式不变

变量设置（以 `15g` 粉量为例）：

| 组别 | 粉水比 | 总水量 |
|------|--------|--------|
| B1 | `1:15` | `225g` |
| B2 | `1:16` | `240g` |
| B3 | `1:17` | `255g` |

重点观察：

- 浓度是否更合你口味
- 厚度、甜感、干净度谁更平衡
- 杯子是“更集中”还是“更空”

常见趋势：

| 方向 | 可能表现 |
|------|---------|
| 比例更小（更浓） | 厚度更高、风味更集中，但可能显闷 |
| 比例更大（更淡） | 更轻盈、更清楚，但可能显薄 |

这一轮的目标是找到你喜欢的“杯子浓度”，不是单纯追求高分。

### 3. 实验 C：水温

最后再看水温怎么影响萃取效率和风味展开。

固定不变：

- 研磨度使用实验 A 的较优刻度
- 粉水比使用实验 B 的较优比例
- 注水方式不变

变量设置：

| 组别 | 水温 |
|------|------|
| C1 | `88C` |
| C2 | `92C` |
| C3 | `96C` |

重点观察：

- 香气是否更打开
- 酸甜是否更完整
- 苦味、木质感、干涩是否被放大

常见趋势：

| 方向 | 可能表现 |
|------|---------|
| 更低 | 更柔和，但可能萃取不足 |
| 更高 | 更容易萃出甜感和厚度，但也更容易带出苦涩 |

如果你的豆子明显偏深，这一轮可以改成 `86C / 90C / 94C`。

---

## 六、第二阶段：高影响的手法变量

### 4. 实验 D：注水结构 / 段数

当 `研磨度 + 粉水比 + 水温` 比较稳定后，再研究注水结构，你会更容易知道风味变化来自哪里。

固定不变：

- 使用实验 A-C 找到的较优组合
- 注水速度尽量保持一致
- 扰动强度尽量保持一致

变量设置：

| 组别 | 设置 |
|------|------|
| D1 | `1` 刀流 / 少段数 |
| D2 | `3` 段注水 |
| D3 | `4-5` 段注水 |

重点观察：

- 层次是否更清晰
- 杯子是否更均匀还是更零散
- 后段是否更拖、更干或更完整

常见趋势：

| 方向 | 可能表现 |
|------|---------|
| 段数少 | 表现直接、可能更干净，但有时不够完整 |
| 段数多 | 萃取更充分、更均匀，但过头时可能显杂或显闷 |

### 5. 实验 E：扰动强度

这里的扰动包括绕圈范围、注水冲击、摇杯、轻搅拌。

固定不变：

- 使用前面较优组合
- 注水段数固定

变量设置：

| 组别 | 设置 |
|------|------|
| E1 | 几乎不额外扰动 |
| E2 | 轻微绕圈 / 轻微粉床翻动 |
| E3 | 明显扰动，如更大的绕圈或一次摇杯 |

重点观察：

- 甜感和厚度是否提高
- 杯子是否从干净变得模糊
- 是否开始出现干涩、粉感、杂味

常见趋势：

| 方向 | 可能表现 |
|------|---------|
| 扰动更弱 | 更干净，但可能偏薄、偏浅 |
| 扰动更强 | 更容易提升萃取和甜感，但也更容易带出涩感和杂味 |

### 6. 实验 F：总粉量 / 粉层厚度

这个变量很容易被忽略，但它会直接改变粉床厚度和流体路径。  
即使粉水比不变，不同粉量也不是同一杯。

固定不变：

- 粉水比固定，例如 `1:16`
- 研磨、水温、注水结构固定

变量设置：

| 组别 | 粉量 | 总水量 |
|------|------|--------|
| F1 | `12g` | `192g` |
| F2 | `15g` | `240g` |
| F3 | `18g` | `288g` |

重点观察：

- 流速是否变化
- 厚度是否自然提高
- 风味是否更集中，还是更闷

常见趋势：

| 方向 | 可能表现 |
|------|---------|
| 粉层更薄 | 更快、更轻、更容易显空 |
| 粉层更厚 | 更慢、更集中，但可能更难冲透 |

### 7. 实验 G：注水速度 / 流量

注水速度会改变水位高度和粉床扰动，是一个常被低估的变量。

固定不变：

- 研磨、水温、粉水比、段数固定

变量设置：

| 组别 | 设置 |
|------|------|
| G1 | 慢流量 |
| G2 | 中等流量 |
| G3 | 快流量 |

重点观察：

- 水位是否持续偏高
- 萃取是否更均匀还是更容易冲穿
- 杯子是否更厚还是更乱

常见趋势：

| 方向 | 可能表现 |
|------|---------|
| 更慢 | 更温和，可能更干净，但也可能偏浅 |
| 更快 | 更容易推动萃取，但可能造成局部冲击和不均匀 |

### 8. 实验 H：闷蒸时间 / 闷蒸水量

这个变量主要影响前段润湿、排气和后续均匀性。

固定不变：

- 其余参数全部固定

变量设置：

| 组别 | 闷蒸设置 |
|------|---------|
| H1 | `20s`，约 `2` 倍粉量 |
| H2 | `30s`，约 `2.5` 倍粉量 |
| H3 | `45s`，约 `3` 倍粉量 |

重点观察：

- 前段是否更顺畅
- 香气释放是否更完整
- 后段是否更稳定

常见趋势：

| 方向 | 可能表现 |
|------|---------|
| 闷蒸不足 | 排气不完全，后段更容易不稳定 |
| 闷蒸更充分 | 前段更均匀，但过长有时会拖慢整体节奏 |

## 七、第三阶段：器具与水的变量

### 9. 实验 I：滤杯

滤杯会直接改变流速模型和萃取结构，建议在你已有一套稳定手法之后再测。

变量设置示例：

| 组别 | 设置 |
|------|------|
| I1 | 同类锥形滤杯 A |
| I2 | 锥形滤杯 B |
| I3 | 扇形 / 平底滤杯 |

重点观察：

- 干净度与厚度的差异
- 风味是更集中还是更展开
- 流速是否更稳定

### 10. 实验 J：滤纸

滤纸虽然经常被忽略，但它对流速和口感真的有影响。

变量设置示例：

| 组别 | 设置 |
|------|------|
| J1 | 薄滤纸 |
| J2 | 基准滤纸 |
| J3 | 厚滤纸 |

重点观察：

- 流速快慢
- 干净度
- 油脂保留与厚度

### 11. 实验 K：水质 / 矿物质组成

这是一个非常重要但也最容易把实验复杂化的变量。  
建议你先固定一种稳定饮用水，等前面都比较顺手了，再开始研究它。

变量设置示例：

| 组别 | 设置 |
|------|------|
| K1 | 低矿物质软水 |
| K2 | 中等矿物质水 |
| K3 | 偏高矿物质水 |

重点观察：

- 酸质是否更清晰
- 甜感是否更明显
- 杯子是否更立体还是更钝

常见趋势：

| 方向 | 可能表现 |
|------|---------|
| 矿物质较低 | 风味可能更亮、更轻，但也可能显空 |
| 矿物质较高 | 口感更饱满，但有时会压住透明感 |

## 八、第四阶段：细化动作变量

### 12. 实验 L：注水高度 / 水柱集中度

变量设置示例：

| 组别 | 设置 |
|------|------|
| L1 | 低注水、轻柔 |
| L2 | 中等高度 |
| L3 | 偏高注水、冲击更强 |

重点观察：

- 粉床是否更容易翻动
- 杯子是否更容易出现不均匀
- 甜感是否上升，还是杂味更明显

### 13. 实验 M：断水时机 / 水位控制

这个变量常常藏在手法细节里，但它会明显影响整体稳定性。

变量设置示例：

| 组别 | 设置 |
|------|------|
| M1 | 水位快落到底再续水 |
| M2 | 保持中等水位续水 |
| M3 | 全程较高水位续水 |

重点观察：

- 是否更容易产生通道
- 风味是否更完整
- 杯子是否更厚或更糊

## 九、建议研究顺序

### 时间很少时

只做这 `4` 个：

1. 研磨度
2. 粉水比
3. 水温
4. 注水结构

### 时间中等时

做到这 `8` 个：

1. 研磨度
2. 粉水比
3. 水温
4. 注水结构
5. 扰动强度
6. 总粉量
7. 注水速度
8. 闷蒸参数

### 时间充足时

按完整顺序推进：

1. 研磨度
2. 粉水比
3. 水温
4. 注水结构
5. 扰动强度
6. 总粉量
7. 注水速度
8. 闷蒸参数
9. 滤杯
10. 滤纸
11. 水质
12. 注水高度
13. 断水时机

## 十、建议实验节奏

### `4` 周最小闭环

| 周次 | 任务 | 目标 |
|------|------|------|
| 第 `1` 周 | 基准配方重复 `3` 杯 + 研磨度实验 | 找到萃取区间 |
| 第 `2` 周 | 粉水比实验 + 水温实验 | 找到浓度和平衡 |
| 第 `3` 周 | 注水结构 + 扰动强度 | 建立手法理解 |
| 第 `4` 周 | 用最佳组合连续冲 `3` 杯 | 验证稳定性 |

### `8-12` 周完整推进

| 阶段 | 研究变量 |
|------|---------|
| 阶段 `1` | 研磨度、粉水比、水温 |
| 阶段 `2` | 注水结构、扰动强度、总粉量、注水速度、闷蒸参数 |
| 阶段 `3` | 滤杯、滤纸、水质 |
| 阶段 `4` | 注水高度、断水时机等细节动作 |

## 十二、结果判读速查

| 杯中表现 | 更可能的问题 | 第一优先调整 |
|---------|-------------|-------------|
| 酸尖、薄、尾段短 | 萃取偏低 | 先磨细一点 |
| 酸有了，但还是空 | 浓度偏低或萃取稍低 | 先把比例调浓一点，必要时再磨细 |
| 甜感出来了，但尾段发干 | 萃取略高 | 先磨粗一点 |
| 苦、涩、木、口腔发干 | 萃取偏高 | 先磨粗一点或降温 |
| 干净但没有内容 | 浓度太低 | 先把比例从 `1:16` 调到 `1:15.5` 或 `1:15` |
| 太重、发闷、层次糊在一起 | 浓度太高或萃取过头 | 先把比例调淡一点，必要时再磨粗 |

## 十三、这份实验最终要帮你得到什么

不管你研究到第几阶段，最终都要慢慢积累出下面这些判断：

- 这支豆子在我这里，研磨度细到什么程度会开始苦涩。
- 我自己更喜欢哪一类粉水比。
- 这支豆子在什么水温区间最完整。
- 我更适合少段数还是多段数的注水结构。
- 我的手法里，哪些扰动会提升甜感，哪些扰动会带来涩感。
- 换滤杯、滤纸和水以后，这杯为什么会变。

当你能稳定说清这些问题，手冲就不再只是“照着别人的配方做”，而是真正开始变成你自己的系统。
