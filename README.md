## 使用建议

建议先阅读"学习资料总索引"，再按索引中的顺序进入各份文档。  
训练时配合杯测与烘焙记录模板使用，逐步建立自己的烘焙、杯测和修正系统。

### 新手快速上手路径

1. 先看 [生豆采购与练习计划](./生豆采购与练习计划.md)，下单买豆
2. 打印 [新手杯测记录表](./新手杯测记录表_可打印.md) 和 [感官训练记录表](./感官训练记录表_可打印.md)
3. 准备 [标签模板](./标签模板.md) 中的标签贴纸
4. 按 [存放养豆与杯测安排](./存放养豆与杯测安排.md) 中的时间表开始训练
5. 用 [数字化与设备建议](./数字化与设备建议.md) 配置好 HiBean
