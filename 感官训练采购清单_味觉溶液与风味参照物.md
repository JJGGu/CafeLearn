# 感官训练采购清单：味觉溶液、酸质、口腔触感与风味参照物

版本说明：本文件按 **2026-03-10** 可核实的公开资料整理。  
骨架主要基于：

1. `WCR Sensory Lexicon v2.0`
2. `SCA Coffee Taster's Flavor Wheel`
3. `SCA Sensory Skills` 的训练方向

这份文档不是让你买一堆东西堆满桌子，而是让你建立一套能长期复用的职业化感官训练系统。

---

## 一、先说结论：世界级烘焙师该买的不是“香精”，而是 5 套训练能力

你的目标如果真的是冲世界顶级烘焙师，感官训练至少要覆盖这 5 件事：

1. **基础味觉识别**：甜、酸、咸、苦。
2. **酸质细分**：柑橘酸、苹果酸、醋酸感、酒石酸/酒感酸。
3. **口腔触感**：厚度、黏稠感、涩感、干燥感。
4. **风味参照库**：花香、水果、坚果、可可、糖类、香料、发酵、负面风味。
5. **盲测区分能力**：三角杯、差异比较、烘焙缺陷映射。

所以淘宝采购也应该分层，而不是一次性乱买。

---

## 二、采购原则

先把这 6 条记住：

1. 只买 **食品级 / 可食用级**，不要买 `工业级 / 分析纯 / 化学纯`。
2. 风味训练优先买 **真实食材**，不要优先买 `香精 / 精油 / 电子烟香料 / 日化香料`。
3. 酸类先买小包装，`100g` 足够练很久。
4. 先做 **核心 12-18 个参照物**，练熟以后再扩张。
5. 同一类别优先买 **最清晰、最典型** 的样本，不要一次买 8 种相近东西。
6. 所有溶液和参考物都要编号，不要裸练。

### 样品杯使用说明

#### 能否重复使用？
**技术上可以清洗后重复使用，但不建议**，原因如下：

| 因素 | 说明 |
|------|------|
| **材质** | 一次性塑料杯多为 PS 或 PP 材质，反复清洗易磨损 |
| **残留气味** | 咖啡油脂和风味物质容易附着在塑料表面，难以彻底去除 |
| **交叉污染** | 不同样品间可能相互影响，干扰盲测准确性 |
| **成本** | 一次性杯子成本低，重复使用节省有限 |

#### 如果必须重复使用

| 步骤 | 操作 |
|------|------|
| 1. 立即清洗 | 使用后马上用清水冲洗，避免咖啡渍干固 |
| 2. 去油脂 | 用少量洗洁精清洗，去除咖啡油脂 |
| 3. 彻底冲洗 | 用清水冲洗 3-4 次，确保无洗涤剂残留 |
| 4. 晾干/烘干 | 完全干燥后再使用，避免水渍影响 |
| 5. 闻味检查 | 使用前闻一下，确认无异味 |

#### 使用建议

| 场景 | 建议 |
|------|------|
| **正式训练/考试模拟** | 使用新杯子，确保标准一致 |
| **日常练习** | 可以重复使用 2-3 次，但需彻底清洁 |
| **杯测比赛前** | 全部换新，避免任何变量 |

#### 更环保的选择

如果担心浪费，可以考虑：
- 购买 **可重复使用的玻璃小杯**（如白酒杯、小茶杯）
- 使用 **陶瓷小杯**（需统一规格）
- 但注意：**盲测时要避免通过杯子外观、重量区分样品**

## 六、你真正该怎么配溶液

### 1. 一个重要原则

`WCR Sensory Lexicon` 里的很多参考标准，本质上是 **属性识别标准**，不是让你做成“像饮料一样好喝”的东西。  
所以你训练时不追求好喝，追求的是：

1. 属性清不清楚。
2. 强度稳不稳定。
3. 你能不能重复认出来。

### 2. 更推荐的基础训练浓度

如果你目标是职业化，不建议只做一组“1x 很强的标准液”。  
更好的方法是做 **低 / 中 / 高** 三档，这样你练的是识别能力，不是死记刺激感。

| 属性 | 建议浓度 | `500mL` 配方 | 备注 |
|------|---------|-------------|------|
| 甜 `sweet` | 蔗糖 `1.0%` | 蔗糖 `5g` + 水至 `500mL` | 基础甜感 |
| 咸 `salty` | 氯化钠 `0.15%` | 食盐 `0.75g` + 水至 `500mL` | 基础咸感 |
| 酸 `sour` 低 | 柠檬酸 `0.015%` | 柠檬酸 `0.075g` + 水至 `500mL` | 低强度基础酸 |
| 酸 `sour` 中 | 柠檬酸 `0.05%` | 柠檬酸 `0.25g` + 水至 `500mL` | 更容易稳定识别 |
| 苦 `bitter` | 咖啡因 `0.01% / 0.02% / 0.035% / 0.05%` | 依次为 `0.05g / 0.10g / 0.175g / 0.25g` + 水至 `500mL` | 只在你能精确称量时使用 |
| 干燥 `mouth drying` | 明矾 `0.05% / 0.07%` | 依次为 `0.25g / 0.35g` + 水至 `500mL` | 训练发干发紧 |
| 金属 `metallic` | 氯化钾 `0.10%` | 氯化钾 `0.5g` + 水至 `500mL` | 这是常见漏项 |

### 3. 酸质细分参考液

这里重点不是“酸强不强”，而是“酸像什么”。

| 方向 | 参考物 | `500mL` 配法 | 训练重点 |
|------|-------|-------------|---------|
| 柑橘酸 | 柠檬 / 柠檬酸 | 一水柠檬酸 `0.075g`/`0.25g` 或 **无水柠檬酸 `0.07g`/`0.23g`** + 水至 `500mL` | 明亮、直给、上扬 |
| 苹果酸 | 青苹果 / 苹果酸 | 苹果酸 `0.25g` 或 `0.5g` + 水至 `500mL` | 清脆、青苹果、拉长感 |
| 醋酸感 | 白醋 / 苹果醋 | 先用白醋或苹果醋直接嗅闻和微量品尝 | 挥发酸、尖、发酵边界 |
| 酒石酸感 | 葡萄 / 酒石酸 | 酒石酸做低中高三档，配合葡萄汁一起练 | 葡萄、酒感、winey |

### 4. 口感 / 触感训练液

这部分最容易被忽略，但它直接影响你判断咖啡结构。

| 属性 | 训练材料 | 说明 |
|------|---------|------|
| `mouth drying` | 明矾溶液 | 纯干燥、发紧、收敛 |
| 多酚涩感 | 浓红茶 | 更像茶涩、果皮涩、过萃涩 |
| `thickness` 黏稠感 | `5%` 蔗糖溶液、番茄汁 | 更接近液体流变变化 |
| `oily` 油膜感 | `1%` 低脂牛奶、淡奶油 / `half-and-half` | 舌面残留和油膜感 |
| `body/fullness` 饱满度 | 同一支咖啡做浓度阶梯 | 这项最该直接用咖啡练 |

### 5. 安全提醒

1. `咖啡因` 不要手估，不要徒手抓，不要让儿童接触。
2. `醋酸` 新手不要自己买高浓度原料，优先用白醋/苹果醋做参考。
3. `奎宁` 如果买不到明确可食用来源，就不要买。
4. 所有酸类、`明矾`、`氯化钾` 都买 **食品级** 小包装，不要买实验室原料。
5. 所有溶液都贴标签和日期，`72` 小时后原则上重配。

## 十八、材料怎么选：给你明确答案

### 1. 苦味：`咖啡因` 优先，`奎宁` 次之

如果你只买一个苦味标准：

1. **先买咖啡因**
2. 奎宁作为第二标准，拿来练“苦味质量差异”
3. 预算充足时，商业标准如 `FlavorActiV bitter` 更好

结论不是“奎宁不行”，而是：

**对咖啡感官训练，咖啡因更应该排第一。**

### 2. 涩感：`明矾` 和 `单宁` 不是替代关系

这点非常重要。

1. **明矾 `alum`** 训练的是纯 `mouth drying`
2. **单宁酸 / 葡萄籽单宁 / 浓红茶** 训练的是更接近饮料型的多酚涩感

所以你不应该问“明矾和单宁谁更好”，而应该问：

1. 我要练纯干燥感，还是练多酚涩感？
2. 我要练的是结构问题，还是风味边界问题？

### 3. 厚度和油膜感也不能混

1. `thickness` 黏稠感：优先用 `5%` 蔗糖溶液和番茄汁
2. `oily` 油膜感：优先用低脂牛奶和淡奶油 / `half-and-half`
3. `body/fullness` 饱满度：优先用同一支咖啡做浓度阶梯
4. 黄原胶可以用，但更适合做“夸张化训练”，不是第一标准

### 4. 金属感是常见漏项

很多人会练酸、甜、苦、涩，却完全不练 `metallic`。  
但实际杯测里，金属感会直接影响你对缺陷、处理法、储存状态和萃取问题的判断。

家庭训练里最实用的办法就是：

1. **食品级氯化钾 `KCl`**
2. 做 `0.10%` 左右的低强度训练液

## 二十、科学实验安排：标准操作流程（SOP）

### 20.1 每次实验前的通用准备

每次训练开始前，固定执行以下步骤，不要跳过：

1. **漱口**：用常温清水漱口 `2` 次，间隔 `30` 秒，清除口腔残留。
2. **等待 `5` 分钟**：不喝咖啡、不吃东西、不抽烟。
3. **准备中性参照水**：备一杯常温蒸馏水，用于每个样品之间的口腔清洁。
4. **编号盲测杯**：将样品杯背面贴编号，不要让自己看到内容物标签。
5. **记录环境**：在记录表上写下日期、时间、室温（感官受温度影响明显）。
6. **确认溶液温度**：所有训练液统一在 `20-25°C` 室温下使用，避免温度差异干扰判断。

---

### 20.2 实验类型 A：基础味觉强度识别实验

**目标**：能稳定识别甜、酸、咸、苦、干燥、金属六种属性，并能区分低 / 中 / 高三档强度。

**每次实验时长**：`20-25` 分钟

**材料准备**：

| 编号 | 内容 | 浓度 |
|------|------|------|
| A1 | 蔗糖溶液（低） | `0.5%` |
| A2 | 蔗糖溶液（中） | `1.0%` |
| A3 | 蔗糖溶液（高） | `2.0%` |
| B1 | 柠檬酸溶液（低） | `0.015%` |
| B2 | 柠檬酸溶液（中） | `0.05%` |
| B3 | 柠檬酸溶液（高） | `0.10%` |
| C1 | 氯化钠溶液（低） | `0.08%` |
| C2 | 氯化钠溶液（中） | `0.15%` |
| D1 | 咖啡因溶液（低） | `0.01%` |
| D2 | 咖啡因溶液（中） | `0.035%` |
| E1 | 明矾溶液（低） | `0.05%` |
| E2 | 明矾溶液（中） | `0.07%` |
| F1 | 氯化钾溶液 | `0.10%` |

**操作步骤**：

1. 将当天训练的 `6-8` 个样品随机编号（如 `01-08`），不要按顺序排列。
2. 每次品尝约 `5mL`，在口中停留 `5-8` 秒后吐出（训练阶段不需要咽下）。
3. 在记录表中填写：编号 → 你判断的属性 → 你判断的强度（`1-5` 分）。
4. 每个样品之间用清水漱口，等待 `30` 秒再进行下一个。
5. 全部完成后，对照答案，计算正确率。

**评估标准**：

| 正确率 | 水平判断 |
|--------|---------|
| `< 60%` | 需要重复本阶段，不要进入下一阶段 |
| `60-79%` | 基础识别建立，可以开始加入对比训练 |
| `80-89%` | 良好，可以进入酸质细分阶段 |
| `≥ 90%` | 优秀，可以进入进阶训练 |

**记录表格**（每次打印或手写）：

```

-----|----------------|-------------------|---------|--------
01   |                |                   |         |
02   |                |                   |         |
03   |                |                   |         |
04   |                |                   |         |
05   |                |                   |         |
06   |                |                   |         |
07   |                |                   |         |
08   |                |                   |         |

本次正确率：_______%    备注：
```

### 20.3 实验类型 B：酸质细分对比实验

**目标**：能区分柑橘酸、苹果酸、醋酸感、酒石酸四种酸的质感差异，不只是"酸不酸"。

**每次实验时长**：`20-30` 分钟

**材料准备**：

| 编号 | 内容 | 配方 |
|------|------|------|
| S1 | 柠檬酸溶液 | 一水柠檬酸 `0.25g` 或 **无水柠檬酸 `0.23g`** + 水至 `500mL` |
| S2 | 苹果酸溶液 | 苹果酸 `0.25g` + 水至 `500mL` |
| S3 | 白醋稀释液 | 白醋 `5mL` + 水至 `500mL`（约 `0.3%` 醋酸） |
| S4 | 酒石酸溶液 | 酒石酸 `0.25g` + 水至 `500mL` |
| S5 | 新鲜柠檬汁稀释液 | 柠檬汁 `10mL` + 水至 `500mL` |
| S6 | 青苹果汁稀释液 | 青苹果汁 `20mL` + 水至 `500mL` |

**操作步骤**：

1. **第一轮：有标签认知**（前 `2` 次训练）
   - 先看标签，品尝，主动感受每种酸的质感特征，用自己的语言描述。
   - 重点感受：酸的位置（舌尖 / 两侧 / 后段）、持续时间、是否有挥发感、是否有果香。

2. **第二轮：盲测区分**（第 `3` 次起）
   - 随机编号，盲测，填写记录表。
   - 每次至少包含 `4` 种酸，其中 `1-2` 种重复出现（用于检验稳定性）。

3. **第三轮：配对比较**（进阶）
   - 每次取两种酸，做 `A/B` 对比，描述差异点，不只是判断"哪个更酸"。

**关键描述维度**（训练时强制自己回答这 `4` 个问题）：

1. 这个酸的位置在哪里？（舌尖 / 两侧 / 全口 / 后段）
2. 这个酸有没有挥发感？（有 = 醋酸方向；无 = 固定酸方向）
3. 这个酸的余韵是拉长的还是收缩的？（拉长 = 苹果酸；收缩 = 柠檬酸）
4. 这个酸有没有果香伴随？（有 = 真实果汁；无 = 纯化学酸）

### 20.4 实验类型 C：口腔触感结构训练实验

**目标**：能区分干燥感、多酚涩感、油膜感、黏稠感四种口腔触感，并能在咖啡中识别出来。

**每次实验时长**：`20-25` 分钟

**材料准备**：

| 编号 | 内容 | 配方 |
|------|------|------|
| T1 | 明矾溶液（干燥感） | 明矾 `0.25g` + 水至 `500mL` |
| T2 | 浓红茶（多酚涩感） | `1` 包锡兰红茶 + `200mL` 沸水，浸泡 `5` 分钟，冷却至室温 |
| T3 | 低脂牛奶（油膜感） | 直接使用，不稀释 |
| T4 | 蔗糖溶液（黏稠感参照） | 蔗糖 `25g` + 水至 `500mL`（`5%` 浓度） |
| T5 | 番茄汁（自然黏稠感） | 直接使用，不稀释 |
| T6 | 清水（空白对照） | 蒸馏水 |

**操作步骤**：

1. **第一步：建立基准感知**
   - 先品尝清水（T6），感受"零触感"基准。
   - 然后依次品尝 T1 → T2 → T3 → T4 → T5，每次漱口后等待 `1` 分钟。
   - 重点感受：触感发生在哪里、什么时候出现、什么时候消退。

2. **第二步：触感持续时间记录**
   - 品尝后吐出，开始计时，记录触感消退时间（秒）。
   - 这个数据帮助你建立"余韵长度"的量化感知。

3. **第三步：强度评分**
   - 用 `0-5` 分评估每种触感的强度（`0` = 感知不到，`5` = 非常强烈）。

4. **第四步：咖啡映射**（每 `2` 周做一次）
   - 冲一杯你熟悉的咖啡，对照上面的参照物，判断这杯咖啡的触感更接近哪种参照。

**记录表格**：


### 20.5 实验类型 D：风味实物闻香实验

**目标**：建立稳定的嗅觉记忆库，能在咖啡中快速调取对应风味词汇。

**每次实验时长**：`15-20` 分钟

**操作步骤**：

1. **闻香方式**：
   - 将参照物放入密封罐，盖上盖子，摇晃 `3-5` 次，然后打开盖子，将鼻子靠近瓶口（不要把鼻子伸进去），短促吸气 `2-3` 次。
   - 不要长时间连续吸闻，嗅觉会快速疲劳。每个样品之间嗅闻新鲜空气 `30` 秒。

2. **每次训练数量**：
   - 初期：每次 `4-6` 种，不要超过 `8` 种（嗅觉疲劳后判断失准）。
   - 熟练后：可以做 `8-10` 种，但中间必须有 `2-3` 次嗅闻新鲜空气的间隔。

3. **记录方式**：
   - 第一印象词（`1` 秒内说出来的词）。
   - 联想词（这个香气让你想到什么场景、食物、记忆）。
   - 强度评分（`1-5` 分）。

4. **盲闻练习**（第 `3` 周起加入）：
   - 将参照物放入不透明容器，随机编号，盲闻后写下判断，再对答案。

**记录表格**：

```
日期：________  时间：________  训练类别：________

编号 | 第一印象词 | 联想描述 | 强度(1-5) | 正确答案 | 是否正确
-----|----------|---------|----------|---------|--------
F-01 |          |         |          |         |
F-02 |          |         |          |         |
F-03 |          |         |          |         |
F-04 |          |         |          |         |
F-05 |          |         |          |         |
F-06 |          |         |          |         |

本次盲闻正确率：_______%    最难区分的两种：_________ vs _________
今日新建立的嗅觉记忆：
```

### 20.6 实验类型 E：三角杯测实验

**目标**：训练差异识别能力，这是职业杯测的核心技能之一。

**每次实验时长**：`25-35` 分钟

**设置方式**：

- 准备 `3` 杯样品，其中 `2` 杯相同，`1` 杯不同。
- 编号为随机三位数（如 `347`、`821`、`563`），不要用 `1/2/3`。
- 自己设置时，用纸条记录哪杯是"不同"的，翻面放置，测试完再看答案。

**变体设计**（从简单到难）：

| 难度 | 设置方式 |
|------|---------|
| 入门 | 同一支豆，浓度差 `20%`（如 `TDS 1.2%` vs `TDS 1.5%`） |
| 初级 | 同一支豆，不同萃取温度（`88°C` vs `94°C`） |
| 中级 | 同一支豆，不同研磨度（粗 vs 细） |
| 进阶 | 不同处理法的同产区豆（水洗 vs 日晒） |
| 高级 | 不同烘焙度的同一支豆（浅 vs 中浅） |

**操作步骤**：

1. 按顺序品尝三杯，每杯品尝约 `8-10mL`。
2. 先不记录，把三杯都尝一遍，形成整体印象。
3. 再次品尝，开始记录差异点。
4. 指出你认为"不同"的那杯，并写出你判断的依据（不能只说"感觉不一样"）。

**记录表格**：

```
日期：________  时间：________  难度级别：________

三杯编号：_______ / _______ / _______

第一轮印象：
  杯 ___：
  杯 ___：
  杯 ___：

我判断"不同"的是：杯 _______

判断依据（必须写具体差异，不能只写"感觉不同"）：
  酸质差异：
  甜感差异：
  触感差异：
  余韵差异：
  风味差异：

正确答案：杯 _______    是否正确：Y / N

如果判断错误，回顾后分析原因：
```

### 20.7 实验类型 F：烘焙缺陷风味映射实验

**目标**：将负面风味参照物与烘焙过程中的具体问题对应起来。

**每次实验时长**：`20-30` 分钟

**材料准备**：

| 参照物 | 对应烘焙问题 | 准备方式 |
|-------|-----------|---------|
| 纸板 / 旧瓦楞纸 | `papery`，发展不足、低温烘焙 | 撕成小块放密封罐 |
| 原木雪糕棒 | `woody`，生豆老化、发展过度 | 放密封罐 |
| 烤过头的吐司 | 烟熏、焦糊、炭感，烘焙过深 | 自己烤，冷却后放罐 |
| 过浓红茶 | 涩、干、收敛，过萃或烘焙过深 | `1` 包茶 + `100mL` 水，浸泡 `8` 分钟 |
| 苹果醋 | 发酵偏差、醋酸感，处理法问题 | 直接使用 |
| 轻度过熟香蕉 | `fermented`，发酵过度 | 买香蕉放 `3-4` 天至表皮变黑 |

**操作步骤**：

1. 先闻香参照物，建立"这个味道 = 这个问题"的映射。
2. 然后冲一杯你怀疑有某种缺陷的咖啡，对照参照物判断。
3. 记录：这杯咖啡最接近哪种缺陷参照？强度如何？可能的烘焙原因是什么？

**记录表格**：

```
日期：________  时间：________

今日测试咖啡：________ 豆 / ________ 烘焙日期 / ________ 冲煮参数

缺陷检查：
  papery（纸板感）：无 / 轻微 / 明显    强度：___/5
  woody（木质感）：无 / 轻微 / 明显     强度：___/5
  烟熏 / 焦糊感：无 / 轻微 / 明显      强度：___/5
  涩感 / 收敛感：无 / 轻微 / 明显      强度：___/5
  发酵偏差：无 / 轻微 / 明显           强度：___/5

综合判断：
  主要缺陷：
  可能的烘焙原因：
  下次调整方向：
```

## 二十一、12 周细化训练计划

> 本计划基于"版本 A：最低可执行版"设计，每周 `4` 次，每次 `20-30` 分钟。  
> 每次训练前必须执行 **20.1 通用准备流程**。

---

### 第 1 周：建立基准感知

**本周目标**：第一次接触所有基础味觉参照物，不追求正确率，只追求"建立第一印象"。

| 训练次 | 实验类型 | 具体内容 | 材料 |
|--------|---------|---------|------|
| 第 1 次 | 类型 A（有标签） | 依次品尝甜、酸、咸、苦，看标签，主动感受 | 蔗糖中浓、柠檬酸中浓、食盐中浓、咖啡因中浓 |
| 第 2 次 | 类型 C（有标签） | 依次品尝干燥感、多酚涩感、油膜感，看标签 | 明矾液、浓红茶、低脂牛奶 |
| 第 3 次 | 类型 D（有标签） | 闻香：花香 + 柑橘类，看标签，写联想词 | 茉莉花茶、柠檬皮、橙皮 |
| 第 4 次 | 类型 D（有标签） | 闻香：坚果 + 可可类，看标签，写联想词 | 杏仁、榛子、可可粉、黑巧克力 |

**本周记录重点**：每种参照物写 `3` 个以上描述词，不要只写一个词。

---

### 第 2 周：强化基础味觉，引入强度概念

**本周目标**：能区分同一属性的低 / 中 / 高三档强度。

| 训练次 | 实验类型 | 具体内容 | 材料 |
|--------|---------|---------|------|
| 第 1 次 | 类型 A（有标签） | 甜感三档强度对比：`0.5% / 1.0% / 2.0%` 蔗糖 | 蔗糖三档溶液 |
| 第 2 次 | 类型 A（有标签） | 酸感三档强度对比：`0.015% / 0.05% / 0.10%` 柠檬酸 | 柠檬酸三档溶液 |
| 第 3 次 | 类型 A（有标签） | 苦感四档强度对比：`0.01% / 0.02% / 0.035% / 0.05%` 咖啡因 | 咖啡因四档溶液 |
| 第 4 次 | 类型 D（有标签） | 闻香：浆果类 + 糖类，写联想词 | 冻干草莓、冻干蓝莓、蜂蜜、糖蜜 |

**本周记录重点**：强度评分（`1-5` 分），并写出"这个强度让我想到什么咖啡场景"。

---

### 第 3 周：第一次盲测，建立自我评估基准

**本周目标**：第一次做盲测，记录基准正确率，不要因为正确率低而沮丧。

| 训练次 | 实验类型 | 具体内容 | 材料 |
|--------|---------|---------|------|
| 第 1 次 | 类型 A（盲测） | 盲测：甜 / 酸 / 咸 / 苦 / 干燥 / 金属，各 `1` 杯，共 `6` 杯 | 六种属性中浓溶液 |
| 第 2 次 | 类型 B（有标签） | 酸质细分第一次接触：柠檬酸 vs 苹果酸，看标签，主动感受差异 | 柠檬酸液、苹果酸液 |
| 第 3 次 | 类型 D（盲闻） | 盲闻：花香 + 柑橘 + 坚果混合，共 `6` 种 | 茉莉、柠檬皮、橙皮、杏仁、榛子、可可粉 |
| 第 4 次 | 类型 E（入门） | 第一次三角杯测：同一支豆，浓度差 `20%` | 同一支咖啡，两种浓度 |

**本周记录重点**：记录第一次盲测正确率，作为后续 `12` 周的基准数据。

---

### 第 4 周：酸质细分深化

**本周目标**：能稳定区分柠檬酸和苹果酸，初步接触醋酸感和酒石酸。

| 训练次 | 实验类型 | 具体内容 | 材料 |
|--------|---------|---------|------|
| 第 1 次 | 类型 B（有标签） | 四种酸依次品尝，看标签，重点感受质感差异 | 柠檬酸、苹果酸、白醋稀释液、酒石酸 |
| 第 2 次 | 类型 B（盲测） | 盲测四种酸，记录正确率 | 四种酸溶液随机编号 |
| 第 3 次 | 类型 C（有标签） | 口腔触感四种对比：干燥 / 多酚涩 / 油膜 / 黏稠 | 明矾液、浓红茶、低脂牛奶、`5%` 蔗糖液 |
| 第 4 次 | 类型 E（初级） | 三角杯测：同一支豆，不同萃取温度 | 同一支豆，`88°C` vs `94°C` |

**本周记录重点**：酸质细分记录表，重点写"挥发感"和"余韵方向"两个维度。

---

### 第 5 周：花香与柑橘风味库建立

**本周目标**：建立花香类和柑橘类的稳定嗅觉记忆。

| 训练次 | 实验类型 | 具体内容 | 材料 |
|--------|---------|---------|------|
| 第 1 次 | 类型 D（有标签） | 花香类全覆盖：茉莉 / 玫瑰 / 桂花，逐一闻香，写描述词 | 茉莉花茶、玫瑰花茶、桂花 |
| 第 2 次 | 类型 D（有标签） | 柑橘类全覆盖：柠檬皮 / 橙皮 / 西柚皮 / 伯爵茶 | 新鲜柠檬皮、橙皮、西柚皮、伯爵茶 |
| 第 3 次 | 类型 D（盲闻） | 盲闻：花香 + 柑橘混合 `6` 种 | 随机编号 |
| 第 4 次 | 类型 A（盲测） | 复习：基础味觉盲测，目标正确率 `≥ 80%` | 六种属性溶液 |

**本周记录重点**：花香和柑橘的区别描述词，至少各写 `5` 个不同的词。

---

### 第 6 周：苹果 / 梨 / 核果与浆果风味库建立

**本周目标**：建立苹果 / 梨 / 核果类和浆果类的稳定嗅觉记忆。

| 训练次 | 实验类型 | 具体内容 | 材料 |
|--------|---------|---------|------|
| 第 1 次 | 类型 D（有标签） | 苹果 / 梨 / 核果类：青苹果 / 红苹果 / 梨 / 杏脯 | 新鲜青苹果、梨、杏脯 |
| 第 2 次 | 类型 D（有标签） | 浆果类：冻干草莓 / 冻干蓝莓 / 黑加仑 / 蔓越莓 | 各类冻干浆果 |
| 第 3 次 | 类型 B（盲测） | 酸质细分盲测，目标正确率 `≥ 70%` | 四种酸溶液 |
| 第 4 次 | 类型 E（中级） | 三角杯测：同一支豆，不同研磨度 | 同一支豆，粗研磨 vs 细研磨 |

**本周记录重点**：浆果类和核果类的差异描述，重点写"甜感类型"和"酸感类型"。

---

### 第 7 周：坚果 / 可可 / 糖类风味库建立

**本周目标**：建立坚果、可可、糖类的稳定嗅觉和味觉记忆。

| 训练次 | 实验类型 | 具体内容 | 材料 |
|--------|---------|---------|------|
| 第 1 次 | 类型 D（有标签） | 坚果类：杏仁 / 榛子 / 花生，闻香 + 品尝 | 原味杏仁、榛子、花生 |
| 第 2 次 | 类型 D（有标签） | 可可类：可可粉 / 可可碎粒 / `85%` 黑巧克力，闻香 + 品尝 | 天然可可粉、黑巧克力 |
| 第 3 次 | 类型 D（有标签） | 糖类：蜂蜜 / 红糖 / 糖蜜 / 焦糖酱，闻香 + 品尝 | 各类糖 |
| 第 4 次 | 类型 C（盲测） | 口腔触感盲测：干燥 / 多酚涩 / 油膜 / 黏稠，随机编号 | 四种触感参照物 |

**本周记录重点**：坚果和可可的"烘烤程度"描述，糖类的"甜感深度"描述。

---

### 第 8 周：香料 / 谷物 / 烘烤类风味库建立 + 阶段性综合盲测

**本周目标**：完成正向风味库建立，做第一次综合盲测评估。

| 训练次 | 实验类型 | 具体内容 | 材料 |
|--------|---------|---------|------|
| 第 1 次 | 类型 D（有标签） | 香料类：丁香 / 肉桂 / 香草 / 黑胡椒 | 各类香料 |
| 第 2 次 | 类型 D（有标签） | 谷物 / 烘烤类：大麦茶 / 爆米花 / 烤吐司 | 大麦茶、爆米花、烤吐司 |
| 第 3 次 | 类型 D（综合盲闻） | 综合盲闻：从已建立的风味库中随机抽取 `10` 种 | 随机 `10` 种参照物 |
| 第 4 次 | 类型 A + B（综合盲测） | 基础味觉 `6` 种 + 酸质细分 `4` 种，共 `10` 杯盲测 | 所有溶液 |

**本周记录重点**：综合盲测正确率，与第 `3` 周基准数据对比，记录进步幅度。

---

### 第 9 周：负面风味参照建立

**本周目标**：建立负面风味的识别边界，能区分"风味弱"和"有缺陷"。

| 训练次 | 实验类型 | 具体内容 | 材料 |
|--------|---------|---------|------|
| 第 1 次 | 类型 F（有标签） | 负面风味参照：纸板 / 木片 / 烤过头吐司，闻香，写描述词 | 纸板、木片、烤过头吐司 |
| 第 2 次 | 类型 F（有标签） | 负面风味参照：过浓红茶 / 苹果醋 / 过熟香蕉，闻香 + 品尝 | 浓红茶、苹果醋、过熟香蕉 |
| 第 3 次 | 类型 F（咖啡映射） | 冲一杯你认为有缺陷的咖啡，对照参照物做缺陷检查 | 有缺陷的咖啡 + 参照物 |
| 第 4 次 | 类型 E（进阶） | 三角杯测：不同处理法的同产区豆 | 水洗 vs 日晒同产区 |

**本周记录重点**：负面风味的"强度边界"——什么程度算轻微，什么程度算明显。

---

### 第 10 周：烘焙映射训练

**本周目标**：能把感官判断和烘焙过程参数对应起来。

| 训练次 | 实验类型 | 具体内容 | 材料 |
|--------|---------|---------|------|
| 第 1 次 | 类型 F（咖啡映射） | 对比同一支豆的浅烘 vs 深烘，做完整缺陷检查 + 风味描述 | 同一支豆两种烘焙度 |
| 第 2 次 | 类型 B + F | 酸质盲测 + 判断这个酸是"漂亮的"还是"发酵偏差" | 酸质溶液 + 苹果醋参照 |
| 第 3 次 | 类型 D（综合盲闻） | 综合盲闻 `10` 种，目标正确率 `≥ 80%` | 随机 `10` 种参照物 |
| 第 4 次 | 类型 E（高级） | 三角杯测：不同烘焙度的同一支豆 | 同一支豆，浅 vs 中浅 |

**本周记录重点**：每次训练后写"这个感官判断对应的烘焙调整方向是什么"。

---

### 第 11 周：连续盲测稳定性训练

**本周目标**：验证感官判断的稳定性，同一个样品在不同时间是否能得出相同结论。

| 训练次 | 实验类型 | 具体内容 | 材料 |
|--------|---------|---------|------|
| 第 1 次 | 类型 A（盲测） | 基础味觉盲测，目标正确率 `≥ 90%` | 六种属性溶液 |
| 第 2 次 | 类型 B（盲测） | 酸质细分盲测，目标正确率 `≥ 80%` | 四种酸溶液 |
| 第 3 次 | 类型 D（盲闻） | 综合盲闻 `12` 种，目标正确率 `≥ 75%` | 随机 `12` 种参照物 |
| 第 4 次 | 类型 E（进阶） | 三角杯测 `×2` 组，连续做，记录疲劳后的判断准确率变化 | 两组不同的三角杯测 |

**本周记录重点**：连续训练中，第几个样品开始出现判断偏差，记录自己的感官疲劳规律。

---

### 第 12 周：综合评估与体系建立

**本周目标**：做完整的综合评估，建立个人感官档案，制定下一阶段训练方向。

| 训练次 | 实验类型 | 具体内容 | 材料 |
|--------|---------|---------|------|
| 第 1 次 | 类型 A + B（综合盲测） | 基础味觉 `6` 种 + 酸质 `4` 种，共 `10` 杯，记录最终正确率 | 所有溶液 |
| 第 2 次 | 类型 C（综合盲测） | 口腔触感 `5` 种盲测，记录最终正确率 | 五种触感参照物 |
| 第 3 次 | 类型 D（综合盲闻） | 综合盲闻 `15` 种，记录最终正确率 | 随机 `15` 种参照物 |
