# 感官训练采购清单：味觉溶液、酸质、口腔触感与风味参照物

版本说明：本文件按 **2026-03-10** 可核实的公开资料整理。  
骨架主要基于：

1. `WCR Sensory Lexicon v2.0`
2. `SCA Coffee Taster's Flavor Wheel`
3. `SCA Sensory Skills` 的训练方向

这份文档不是让你买一堆东西堆满桌子，而是让你建立一套能长期复用的职业化感官训练系统。

---

## 一、先说结论：世界级烘焙师该买的不是“香精”，而是 5 套训练能力

你的目标如果真的是冲世界顶级烘焙师，感官训练至少要覆盖这 5 件事：

1. **基础味觉识别**：甜、酸、咸、苦。
2. **酸质细分**：柑橘酸、苹果酸、醋酸感、酒石酸/酒感酸。
3. **口腔触感**：厚度、黏稠感、涩感、干燥感。
4. **风味参照库**：花香、水果、坚果、可可、糖类、香料、发酵、负面风味。
5. **盲测区分能力**：三角杯、差异比较、烘焙缺陷映射。

所以淘宝采购也应该分层，而不是一次性乱买。

---

## 二、采购原则

先把这 6 条记住：

1. 只买 **食品级 / 可食用级**，不要买 `工业级 / 分析纯 / 化学纯`。
2. 风味训练优先买 **真实食材**，不要优先买 `香精 / 精油 / 电子烟香料 / 日化香料`。
3. 酸类先买小包装，`100g` 足够练很久。
4. 先做 **核心 12-18 个参照物**，练熟以后再扩张。
5. 同一类别优先买 **最清晰、最典型** 的样本，不要一次买 8 种相近东西。
6. 所有溶液和参考物都要编号，不要裸练。

## 六、你真正该怎么配溶液

### 1. 一个重要原则

`WCR Sensory Lexicon` 里的很多参考标准，本质上是 **属性识别标准**，不是让你做成“像饮料一样好喝”的东西。  
所以你训练时不追求好喝，追求的是：

1. 属性清不清楚。
2. 强度稳不稳定。
3. 你能不能重复认出来。

### 2. 基础味觉 1x 参考液

下面这些浓度适合做“标准参考液”。

| 属性 | 官方参考方向 | `500mL` 配方 | 备注 |
|------|-------------|-------------|------|
| 甜 `sweet` | 蔗糖 `12.0%` | 蔗糖 `60g` + 水至 `500mL` | 很甜，但识别清楚 |
| 咸 `salty` | 氯化钠 `0.08%` | 食盐 `0.4g` + 水至 `500mL` | 用来校准咸感 |
| 酸 `sour` | 柠檬酸 `0.15%` | 柠檬酸 `0.75g` + 水至 `500mL` | 基础酸感标准 |
| 苦 `bitter` | 咖啡因 `0.05%` | 咖啡因 `0.25g` + 水至 `500mL` | 只在你能精确称量时使用 |
| 厚度 `fullness` | 奶油粉 `5.0%` | 奶油粉 `25g` + 水至 `500mL` | 训练厚度 |
| 黏稠 `thickness` | 黄原胶 `0.8%` | 黄原胶 `4g` + 水至 `500mL` | 需充分搅拌、静置后再练 |

### 3. 酸质细分 1x 参考液

这里重点不是“酸强不强”，而是“酸像什么”。

| 方向 | 参考物 | `500mL` 配法 | 训练重点 |
|------|-------|-------------|---------|
| 柑橘酸 | 柠檬 / 柠檬酸 | 柠檬酸 `0.75g` + 水至 `500mL` | 明亮、直给、上扬 |
| 苹果酸 | 青苹果 / 苹果酸 | 苹果酸 `5g` + 水至 `500mL` | 清脆、青苹果、拉长感 |
| 醋酸感 | 白醋 / 苹果醋 | 先用白醋或苹果醋直接嗅闻和微量品尝 | 挥发酸、尖、发酵边界 |
| 酒石酸感 | 葡萄 / 酒石酸 | 酒石酸 `0.4g` + 水至 `500mL` | 葡萄、酒感、winey |

### 4. 强度训练不要重新称量，直接稀释

做完 `1x` 参考液后，直接做 3 个强度级别：

1. `低`：`25mL` 参考液 + `75mL` 水
2. `中`：`50mL` 参考液 + `50mL` 水
3. `高`：`75mL` 参考液 + `25mL` 水

这样你的训练重点会从“称量误差”转回“识别能力”。

### 5. 安全提醒

1. `咖啡因` 不要手估，不要徒手抓，不要让儿童接触。
2. `醋酸` 新手不要自己买高浓度原料，优先用白醋/苹果醋做参考。
3. 所有酸类都买 **食品级** 小包装，不要买实验室原料。
4. 所有溶液都贴标签和日期，`72` 小时后原则上重配。

## 九、一个真正适合烘焙师的每周训练结构

### 版本 A：最低可执行版

每周 `4` 次，每次 `20-30` 分钟：

1. `1` 次基础味觉 + 酸质盲测
2. `1` 次口腔触感训练
3. `1` 次风味实物闻香
4. `1` 次三角杯或咖啡对照杯测

### 版本 B：职业进阶版

每周 `6` 次：

1. `2` 次味觉 / 酸质 / 口腔触感
2. `2` 次风味库闻香与盲闻
3. `1` 次咖啡三角杯
4. `1` 次烘焙缺陷样对照杯测

## 十、12 周训练建议

### 第 1-2 周：只练“基础味觉和强度”

只做：

1. 甜
2. 酸
3. 咸
4. 苦
5. 厚度
6. 黏稠感

目标不是词汇量，而是稳定识别。

### 第 3-4 周：加入酸质细分

重点区分：

1. 柑橘酸
2. 苹果酸
3. 醋酸感
4. 酒石酸 / winey 感

### 第 5-8 周：建立正向风味库

按下面顺序：

1. 花香
2. 柑橘
3. 苹果 / 梨 / 核果
4. 浆果
5. 坚果 / 可可
6. 糖类 / 褐变

### 第 9-12 周：加入负面参照和烘焙映射

训练你回答这 3 个问题：

1. 这是不是风味弱，而不是烘坏？
2. 这是木、纸、烟、焦，还是只是发展偏深？
3. 这个酸是漂亮的，还是发酵偏差？

## 十七、最后给你的实话

如果你真想冲世界级，感官训练不要停留在“我能说出巧克力、榛子、花香”。  
真正的职业标准是：

1. 你能不能把这些感觉稳定地重复认出来。
2. 你能不能把差异和烘焙过程对应起来。
3. 你能不能在盲测里连续说对。

做到这一步，你的感官才开始真正服务烘焙，而不是只服务描述。
