# CVA 杯测与感官校准基础

版本说明：本文件按 **2026-03-09** 可核实的公开资料整理，核心基于 `SCA` 现行 `Coffee Value Assessment (CVA)` 逻辑。  
这份文档的目的不是代替正式课程，而是让你在日常烘焙训练里，尽快用上正确的现代杯测框架。

---

## 一、为什么你必须学 CVA

你现在最大的痛点之一是：

1. 喝不清楚。
2. 说不清楚。
3. 说清楚了也不一定知道怎么改。

`CVA` 的价值就在这里。  
它帮你把一杯咖啡拆成不同维度，而不是用一个模糊总分去概括全部。

### 旧思路的问题

很多烘焙师的杯测还停留在一种旧习惯：

1. 先判断喜不喜欢。
2. 再补几个风味词。
3. 最后给一个笼统结论。

这种方式的问题是：

1. 主观喜好和客观描述混在一起。
2. 你很难和别人校准。
3. 你很难追踪自己到底哪里进步了。

### CVA 的核心价值

`CVA` 更强调：

1. 把“描述”与“喜欢”分开。
2. 把“物理信息”和“杯中信息”联系起来。
3. 把“咖啡本身是什么”与“我喜不喜欢它”分开。
4. 让你在不同语境下表达咖啡价值。

对烘焙师来说，最大的实际意义是：

**你终于能把杯测从“感想”升级成“诊断工具”。**

## 二、CVA 的四个核心评估维度

截至 **2026-03-09**，SCA 教育页面与 Q 备考页面反复强调四个重要维度：

1. `Physical Assessment`
2. `Descriptive Assessment`
3. `Affective Assessment`
4. `Extrinsic Assessment`

你需要分别理解。

## 1. Physical Assessment

这是对咖啡物理层面的评估。  
在 `Q Grader` 准备页面里，SCA 直接把它称为 “green grading”。

对烘焙师，它意味着：

1. 生豆外观缺陷不只是视觉问题。
2. 含水率、豆形、大小、均匀度会影响烘焙表现。
3. 你要先知道原料物理状态，再谈杯中结果。

日常烘焙中你至少要关注：

1. 缺陷豆比例。
2. 豆目大小差异。
3. 密度和含水率的间接表现。
4. 新鲜度和储存状态。

## 2. Descriptive Assessment

这是描述性评估。  
你要回答的是：

1. 这杯闻起来像什么。
2. 喝起来像什么。
3. 强度有多大。
4. 结构呈现得是否清楚。

这里强调的是“描述”，不是“喜欢”。

正确例子：

1. 柑橘类酸感，中等偏高。
2. 蜂蜜甜，中等。
3. 茉莉花和白桃香气，热杯清晰，冷却后转向柑橘皮。
4. 中段偏薄，尾段有轻微木感。

错误例子：

1. 很高级。
2. 不错。
3. 这杯我喜欢。

## 3. Affective Assessment

这是喜好与偏好维度。  
它承认一个现实：

1. 描述是相对共享的。
2. 喜好是有主观性的。

你在职业训练中必须学会：

1. 我是否喜欢这杯。
2. 我喜欢的是哪种价值。
3. 我不喜欢，是因为质量问题，还是因为风格不合我偏好。

举例：

1. 一支风格明亮、酸高、结构轻盈的豆子，我个人未必最喜欢，但它依然可能非常干净、清晰且高质量。
2. 一支甜感厚重、巧克力坚果主导的豆子，我可能更喜欢喝，但它未必在描述层面更复杂。

## 4. Extrinsic Assessment

这是外部信息维度。  
例如：

1. 产地。
2. 处理法。
3. 品种。
4. 海拔。
5. 品牌或故事信息。
6. 价格、稀缺性、可持续信息。

它的意义是：

咖啡的“价值”不只有杯中风味。  
但你必须先把杯中信息和外部信息分开，再去讨论整体价值。

对烘焙师尤其重要，因为：

1. 你不能因为一支豆故事很好，就放松对杯中缺陷的判断。
2. 你也不能因为一支豆风味不张扬，就忽略它在市场、稳定性或适用场景上的价值。

## 三、CVA 背后的核心逻辑

## 1. 多视角评价

官方 `Q Grader` 页面明确提到：  
新的 `Q` 强调 “multi-perspective evaluation”，也就是多视角评价。

这意味着你必须学会区分：

1. 这杯是什么。
2. 这杯表现得是否清晰。
3. 我个人是否喜欢。
4. 它在某个市场里是否有高价值。

如果你把这四件事混在一起，你的烘焙修正就会经常出错。

## 2. 共享语言

`CVA` 不只是表格，更是一套共享语言。  
职业化的重点不是你脑子里有多丰富的词，而是：

1. 你用的词是否稳定。
2. 你和别人是否能校准。
3. 这个词是否真的对应可感知的东西。

## 3. 主观和客观尽量分层

感官工作不可能完全客观，但可以更可重复。  
所以你应该在记录里分成两列：

1. 我观察到了什么。
2. 我对它的偏好是什么。

## 四、CVA 里你必须掌握的工具语言

根据 SCA 官方课程说明，`CVA for Cuppers` 会训练学员使用：

1. 15-point intensity scales
2. CATA lists
3. hedonic scales

你必须理解这三个东西。

## 1. 15 点强度尺度

它的目的不是让你“打高分”，而是估计某种属性有多强。  
这种强度尺度非常适合烘焙训练，因为你可以追踪：

1. 某支豆的柑橘酸强度有没有变化。
2. 花香从样品 A 到 B 是增强还是减弱。
3. 苦感或口腔干燥感是不是因为发展变化而提升。

你不需要一开始就像专业感官面板一样精准，但要逐步做到：

1. 同一属性在不同样品间能分出高低。
2. 同样样品在不同天喝，不会乱飘。
3. 你和其他人打出来的大方向接近。

## 2. CATA 列表

`CATA` 是 `Check-All-That-Apply`。  
意思是从标准化列表中勾选适用的描述词。

优点：

1. 降低自由发挥带来的混乱。
2. 更容易和别人对照。
3. 更适合训练初中级描述稳定性。

缺点：

1. 容易把人困在列表里。
2. 容易只会“勾词”，不会真正感知强度和结构。

正确用法：

1. 先用列表校准。
2. 再补充自由描述。
3. 最终形成自己稳定但能共享的语言。

## 3. Hedonic scales

这是喜好尺度。  
你要表达的是“喜欢程度”，不是“质量程度”。

烘焙师经常犯的错误是：

1. 喜欢深一点的口味，就把深烘样品一概认为更好。
2. 喜欢花果调性，就把厚甜豆一概看低。

职业化判断必须能说清：

1. 这杯我个人偏好如何。
2. 这杯在目标市场里会不会被喜欢。
3. 这杯是否完成了它应完成的风味表达。

## 五、你在日常训练里该怎样使用 CVA

## 1. 每次杯测都分成三层记录

### 第一层：描述层

写：

1. 香气类型。
2. 酸质类型与强度。
3. 甜感类型与强度。
4. 口感。
5. 结构完整度。
6. 尾段干净度。
7. 负面特征。

### 第二层：喜好层

写：

1. 我喜不喜欢。
2. 喜欢在哪里。
3. 不喜欢在哪里。

### 第三层：烘焙诊断层

写：

1. 我认为问题更来自前段、中段还是发展段。
2. 下一炉只改什么。

这第三层，就是让 `CVA` 真正服务于烘焙。

## 2. 用热杯、温杯、凉杯三次观察

很多人只喝热杯就下结论。  
这是不够的。

因为：

1. 热杯更容易强调挥发性香气。
2. 温杯更能看结构和平衡。
3. 凉杯更容易暴露负面问题和后段干净度。

你以后每次正式杯测，至少分三段记。

## 3. 同一批实验样品必须在相同养豆天数比较

否则你的结论会混乱。  
如果样品 A 是第 4 天，样品 B 是第 10 天，你其实在比较两件事：

1. 烘焙差异。
2. 养豆差异。

这会让你误判。

截至 **2026-03-09**，SCA 官方 `Q Grader` 准备页说明，书面考试重点包括：

1. `CVA` 协议。
2. `SCA-102`、`SCA-103`、`SCA-104`。

这意味着你至少要非常清楚：

1. 样品如何准备。
2. 描述、喜好、物理评估各自是什么。
3. 哪些内容属于 protocol。
4. 哪些属于表单应用。

## 七、标准样品烘焙与杯测烘度

`Q Grader` 准备页给出了一个非常重要的信息：  
为了准备 `Roasting Problems Practical Exam`，需要熟悉标准杯测烘焙与错误烘焙。

官方页面给出的训练基准是：

1. 正常标准杯测烘焙：磨粉 `Agtron 63`，约等于整豆 `Agtron 58`
2. 过浅：磨粉约 `75`，整豆约 `68`
3. 过深：磨粉约 `55`，整豆约 `45`
4. 过快：达到标准颜色，但时间约为正常一半
5. 过慢：达到标准颜色，但时间约为正常两倍

这对你有两个非常实用的启发：

1. 焙度颜色和烘焙速度不是同一件事。
2. 同样颜色也可能对应完全不同的风味质量。

这也是很多家用烘焙师最容易忽视的地方。

## 八、感官训练里你必须建立的五种能力

## 1. 识别能力

你能不能识别：

1. 主香气方向。
2. 酸质方向。
3. 甜感方向。
4. 负面问题。

## 2. 强度判断能力

你能不能判断：

1. 这个属性强还是弱。
2. 比另一杯强多少。

## 3. 差异判断能力

你能不能在三角杯里稳定找出不同的一杯。

## 4. 一致性能力

你今天和明天对同一杯的判断会不会漂移很大。

## 5. 语言输出能力

你能不能把感觉准确说出来，而不是只说：

1. 好喝。
2. 不好喝。
3. 有点怪。

---

## 九、你最容易犯的感官错误

1. 把高酸误当成高质量。
2. 把甜重误当成发展充分。
3. 把涩感误当成明亮感。
4. 把焙火味误当成厚度。
5. 把发酵香和脏感混为一谈。
6. 把自己喜欢的风格直接等同于客观更好。
7. 只闻到一个词就立刻下结论。
8. 用风味词逃避结构判断。

风味词只是入口，不是全部。  
职业烘焙师更要关心：

1. 清晰度。
2. 平衡。
3. 连贯性。
4. 后段。
5. 缺陷暴露。

## 十、你如何把 WCR Lexicon 和 Flavor Wheel 用起来

`SCA` 的 `Coffee Taster’s Flavor Wheel` 是和 `WCR` 合作更新的，底层基础来自 `WCR Sensory Lexicon`。  
你必须知道两者差异：

### Flavor Wheel 是什么

它更像：

1. 词汇导航图。
2. 训练描述方向的工具。

适合做：

1. 从大类走到小类。
2. 从模糊印象走到更精确表达。

### Lexicon 是什么

它更像：

1. 科学化描述字典。
2. 带定义与参考物的训练系统。

适合做：

1. 训练词义边界。
2. 建立参考物。
3. 训练强度判断。

你作为烘焙师，正确的用法是：

1. 用 `Flavor Wheel` 帮自己找到方向。
2. 用 `Lexicon` 训练稳定定义和参考。

## 十一、把 CVA 落到你的训练里

建议你每周至少做以下四类训练：

## 1. 一次正式盲测

要求：

1. 样品编号。
2. 至少两杯重复杯。
3. 分热、温、凉记录。

## 2. 一次三角杯

目的：

1. 训练差异识别。
2. 训练不要被预设影响。

## 3. 一次味觉基础溶液训练

练习：

1. 甜。
2. 酸。
3. 咸。

这是 `Q Grader` 准备页直接提到的 `Main Taste Solutions` 方向。

## 4. 一次香气参考训练

用：

1. 花香。
2. 柑橘。
3. 坚果。
4. 巧克力。
5. 发酵。
6. 木纸。

去训练“先闻参考物，再闻咖啡”的映射。

## 十二、你现在的最小执行版本

如果你不想一开始把体系搞太复杂，就先做这个最小版本：

1. 每次杯测必须分“描述”和“喜好”两栏。
2. 每次杯测必须写热杯、温杯、凉杯。
3. 每次杯测必须只给下一炉一个改动建议。
4. 每周必须做一次三角杯。
5. 每周必须做一次基础味觉练习。

只要你把这五件事坚持 8 周，你的风味识别能力会明显提升。
