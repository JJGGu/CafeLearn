# SCA 学习框架与课程地图

版本说明：本文件按 **2026-03-09** 可核实的官方信息整理。  
重要日期：**2025-10-01** 起，`Q Grader` 项目由 `SCA` 运营并建立在 `CVA` 体系之上；因此你如果看到大量旧的 `CQI Arabica Q` 备考资料，需要带着“新旧体系并不完全相同”的意识去看。

---

## 一、这份文档解决什么问题

你现在已经有一份自己的训练计划，但要走职业化路线，只靠“练习框架”还不够。  
你还需要知道：

1. `SCA` 现在到底有哪些学习体系。
2. 哪些体系和烘焙师真正相关。
3. `SCA`、`CVA`、`Q Grader` 之间是什么关系。
4. 你应该按什么顺序学，才能不重复投入时间。

这份文档就是你的“地图”。  
后面几份文档则是每一个模块的“知识正文”。

## 二、先把几个概念分清

### 1. SCA 是什么

`SCA` 是 `Specialty Coffee Association`，也就是精品咖啡协会。  
它不只是办比赛，也不只是卖证书。它还做：

1. 教育课程。
2. 标准制定。
3. 感官与杯测体系更新。
4. 研究传播。
5. 行业职业资格体系。

对你而言，`SCA` 重要的不是“证书名气”，而是它把很多行业实践标准化了。

### 2. CSP 是什么

`CSP` 是 `Coffee Skills Program`，也就是 `SCA` 当前的模块化课程体系。  
截至 **2026-03-09**，官方页面明确写的是：

1. 先修 `Introduction to Coffee`。
2. 再进入五个模块中的一个或多个。
3. 每个模块有 `Foundation / Intermediate / Professional` 三个层级。
4. 课程通过后取得证书，并可累积到 `Skills Diploma`。

### 3. CVA 是什么

`CVA` 是 `Coffee Value Assessment`。  
这是现在 `SCA` 用来更新杯测与价值评估逻辑的核心框架。

它和旧式只追求“打分”的理解不同。`CVA` 更强调：

1. 物理属性。
2. 描述性属性。
3. 喜好性判断。
4. 外部信息。
5. 多视角评价。

对烘焙师来说，`CVA` 的意义非常大，因为它能帮助你把“风味描述”和“主观喜欢”分开。

### 4. Q Grader 是什么

截至 **2026-03-09**，新的 `Q Grader` 由 `SCA` 运营。  
它不是入门课，而是一个高阶职业资格。

官方描述里，`Q Grader` 是：

1. 面向已有杯测和感官分析经验的人。
2. 以 `CVA` 为基础。
3. 强调专业环境中的综合评估能力。
4. 包含书面与多个实作考试。

所以正确关系是：

`SCA`  
→ 提供 `Coffee Skills Program`、`CVA for Cuppers`、`Q Grader` 等教育产品  
→ `CVA` 是新一代杯测和价值评估框架  
→ `Q Grader` 是建立在 `CVA` 上的高级资格

## 三、对烘焙师最相关的 SCA 模块

如果你的目标是成为世界级烘焙师，`SCA` 体系里最相关的不是所有模块平均学，而是按优先级来。

### 第一优先级

1. `Sensory Skills`
2. `Roasting`
3. `Green Coffee`

原因：

1. `Sensory Skills` 决定你能不能正确感知与表达。
2. `Roasting` 决定你能不能把目标转成工艺动作。
3. `Green Coffee` 决定你能不能在烘之前就做出正确判断。

### 第二优先级

1. `Introduction to Coffee`
2. `Coffee Value Assessment (CVA) for Cuppers`
3. `Brewing`

原因：

1. `Introduction to Coffee` 是整体语言校准。
2. `CVA for Cuppers` 能帮助你理解当前行业杯测逻辑。
3. `Brewing` 很重要，因为烘焙质量最终必须通过萃取被稳定表达出来。

### 第三优先级

1. `Barista Skills`
2. `Coffee Technicians Program`
3. `Sustainability Program`

这些不是不重要，而是对你当前阶段的边际收益低于前三类。

## 四、SCA 当前学习结构

## 1. Introduction to Coffee

这是一个基础总览模块。  
官方强调它是从产地到杯中的总体介绍，并已经更新到 `CVA` 的逻辑。

对你真正有用的部分：

1. 咖啡价值链的整体理解。
2. 咖啡新鲜度和基础风味概念。
3. 产地、处理、烘焙、冲煮之间的关系。

对你不够的部分：

1. 不足以支撑专业烘焙判断。
2. 不足以支撑正式感官训练。
3. 不足以支撑生豆采购或质量控制。

结论：

把它当作“行业语言入门”，不要把它当作深度烘焙课程。

## 2. Sensory Skills

官方把这个模块定位为：

1. 感官科学。
2. 咖啡评价。
3. 盲测、测试设计和感官项目管理。

对烘焙师，这是最关键模块之一。  
因为你很多烘焙上的停滞，其实不是工艺问题，而是：

1. 分不清酸、涩、苦、干、空。
2. 说不清“哪里不好”。
3. 不能稳定复现自己的判断。

这个模块的三个层级可以理解为：

### Foundation

重点是：

1. 味觉与嗅觉基础。
2. 基础杯测流程。
3. 风味识别入门。

### Intermediate

重点是：

1. 更复杂的描述。
2. 三角杯与差异判断。
3. 组织感官测试。

### Professional

重点是：

1. 设计感官项目。
2. 训练并校准他人。
3. 在业务环境中使用感官数据。

对于世界级烘焙师，这个模块不是“考一下就完”，而是长期职业能力。

## 3. Roasting

官方把这个模块定位为：

1. 烘焙原理。
2. 曲线与颜色控制。
3. 设备操作。
4. 生产与质量控制。

这个模块和你最直接相关。  
但你要注意，`SCA Roasting` 不是只教“怎么调火”，它本质上教的是：

1. 如何理解热传递。
2. 如何通过过程控制达成目标。
3. 如何建立质量一致性。
4. 如何管理生产环境中的风险与偏差。

三个层级可以这样理解：

### Foundation

重点是：

1. 烘焙时豆子发生了什么变化。
2. 设备基本操作与安全。
3. 基础曲线理解。

### Intermediate

重点是：

1. 更准确地控制 profile。
2. 理解热传递与颜色的关系。
3. 能解释常见烘焙缺陷。

### Professional

重点是：

1. 建立生产 SOP。
2. 在不同豆子、不同目标下设计曲线。
3. 做质量控制和产能管理。

## 4. Green Coffee

官方把它定位为：

1. 生豆评估。
2. 处理与分级。
3. 市场与贸易基础。

你如果只想在家里把豆子烘好，可能会低估这个模块。  
但对于高级烘焙师，`Green Coffee` 非常关键，因为它决定：

1. 你是否知道这支豆为什么会这样表现。
2. 你是否能预测烘焙难点。
3. 你是否能判断问题是来自生豆还是来自烘焙。

三个层级的职业含义大致是：

### Foundation

1. 看懂咖啡基础链路。
2. 认识基本分级与质量判断。

### Intermediate

1. 识别常见缺陷。
2. 理解处理法、分级、品质与价格的关系。

### Professional

1. 样品分析。
2. 合同和采购逻辑。
3. 采购策略与风味目标的结合。

## 5. Brewing

很多烘焙师忽视这个模块，但它对杯测之外的落地非常重要。

你至少要掌握：

1. 萃取率和浓度的基本概念。
2. 水质与温度对表达的影响。
3. 同一支豆为什么在冲煮端会被误判。

因为有些你以为的烘焙问题，实际上是冲煮变量导致的。

## 五、SCA 之外但和 SCA 强相关的两个体系

## 1. CVA for Cuppers

这是一个两天的过渡型课程，适合已经有杯测经验的人。  
截至 **2026-03-09**，官方页面说明：

1. 课程帮助旧系统用户切换到 `CVA`。
2. 课程将于 **2026-11-30** 后停止教授。
3. 它不是“证明你就是专业杯测师”的执照。
4. 它的价值在于让你掌握新工具。

这对你有帮助，因为你现在正处在从“经验型烘焙”走向“系统型烘焙”的阶段。

## 2. Q Grader

这是高阶职业资格。  
它不是先学它再学基础，而是基础、训练和职业经验积累到一定程度之后再去挑战。

如果顺序搞反，你会很痛苦。

## 六、你应该怎么走这条学习路径

下面是更适合你当前状态的顺序。

## 第一阶段：先补底层语言

建议目标：

1. 学完 `Introduction to Coffee` 的核心语言。
2. 系统补 `Sensory Skills Foundation` 水平知识。
3. 补齐最基础的 `Green Coffee` 和 `Roasting Foundation` 知识点。

你当前最缺的是：

1. 术语统一。
2. 风味表达。
3. 生豆到杯中的因果链。

## 第二阶段：让你的感官先追上你的烘焙炉数

建议目标：

1. 把 `Sensory Skills` 作为主线。
2. 同步学习 `Roasting Intermediate` 和 `Green Coffee Intermediate` 的核心知识。
3. 开始用 `CVA` 逻辑写杯测记录。

这阶段你会明显感受到：

1. 以前很多“玄学”其实可以拆解。
2. 以前很多“喝不出来”其实是因为没训练到可复述。

## 第三阶段：从会烘，走向会判断

建议目标：

1. 建立自己的样品评估流程。
2. 能根据生豆信息预判烘焙策略。
3. 能把杯感稳定地映射回过程节点。

这时你才有资格认真准备 `Q Grader`。

## 第四阶段：以 Q 作为综合校验，而不是入门捷径

当你满足以下条件时，可以考虑冲 `Q Grader`：

1. 你经常做正式杯测。
2. 你已经能稳定做三角杯和味觉校准。
3. 你对 `CVA` 表单并不陌生。
4. 你对标准样品烘焙、物理评估和感官描述已经有实际操作经验。

## 七、如果你只想围绕“世界级烘焙师”来学，优先级怎么排

### 必学

1. 感官科学与杯测。
2. 烘焙热力学与工艺控制。
3. 生豆物理属性、处理法、缺陷与储存。
4. 冲煮与萃取基础。
5. 数据记录和质量控制。

### 强烈建议

1. 采购与样品评估。
2. 颜色测量与烘后 QC。
3. 焙度与风味目标管理。
4. 多人校准与感官小组训练。

### 更高阶时再补

1. 生产管理。
2. 成本与损耗管理。
3. 合同与贸易逻辑。
4. 团队培训体系。

## 九、我对你的学习排序建议

如果你未来 12 个月只抓最有效路线，我建议这样：

1. 先把感官和杯测做扎实。
2. 同步补烘焙热传递与阶段控制知识。
3. 再系统补生豆理解。
4. 再把 `CVA` 练熟。
5. 最后再以 `Q Grader` 为综合检验目标。

原因很简单：

一个烘焙师如果感官不清晰，后面所有曲线优化都会很慢。  
一个烘焙师如果生豆理解不足，就会把不该归罪于烘焙的问题归罪于烘焙。  
一个烘焙师如果不了解现行 `CVA`，就会用旧的、不够稳定的方式表达价值。
