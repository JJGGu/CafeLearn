### 新手香气训练方法

#### 第 1-4 周：建立基础参考库

每周选 2 个类别，用以下方法训练：

1. **买实物**：去超市买对应水果/食材
2. **闻实物 → 闻咖啡**：先闻参考物 30 秒，再闻咖啡，看能否找到对应
3. **记录**：写下你闻到的感觉，不要怕写错

#### 第 5-8 周：盲闻训练

1. 把参考物放在不透明杯子里
2. 盲闻判断类别
3. 记录正确率

## 五、新手风味词汇速查表

> 如果你不知道怎么描述，先从这个表里找最接近的词。

### 酸的描述词

| 感觉 | 对应词 | 类比 |
|------|--------|------|
| 明亮、清爽、让人精神一振 | 明亮酸 | 像咬一口青苹果 |
| 柔和、圆润、不刺激 | 圆润酸 | 像成熟的桃子 |
| 尖锐、刺口、不舒服 | 尖酸 | 像没熟的柠檬 |
| 有酒感、有发酵感 | 发酵酸 | 像红酒或醋 |

### 甜的描述词

| 感觉 | 对应词 | 类比 |
|------|--------|------|
| 清爽、轻盈的甜 | 清甜 | 像甘蔗汁 |
| 温暖、厚实的甜 | 蜂蜜甜/红糖甜 | 像蜂蜜水 |
| 烘烤感的甜 | 焦糖甜 | 像焦糖布丁 |
| 水果的甜 | 果甜 | 像熟透的水果 |
| 沉闷、不清爽的甜 | 闷甜 | 像过熟的香蕉 |

### 口感描述词

| 感觉 | 对应词 | 类比 |
|------|--------|------|
| 像水一样薄 | 薄/水感 | 像淡茶 |
| 有一定厚度 | 中等body | 像全脂牛奶 |
| 很厚实 | 厚/奶油感 | 像奶昔 |
| 嘴巴发干发紧 | 涩/干燥 | 像浓茶的涩感 |
| 滑顺不粗糙 | 丝滑 | 像丝绸 |
| 粗糙刮舌 | 粗糙/砂感 | 像细砂纸 |

### 负面问题描述词

| 感觉 | 对应词 | 可能原因 |
|------|--------|---------|
| 像没熟的蔬菜 | 生青 | 发展不足 |
| 像旧纸箱 | 纸板/木质 | 生豆老化或梅纳不足 |
| 像烟灰 | 烟感 | 烘焙过程污染或过深 |
| 像烧焦的东西 | 焦苦 | 过度发展或局部过热 |
| 香气闷在里面出不来 | 发闷 | 发展过度或过慢 |
| 中间空空的没内容 | 空洞 | 中段能量不足 |
| 嘴巴发干 | 干涩 | 一爆前冲刺过猛 |
| 有金属味 | 金属感 | 萃取问题或生豆问题 |
