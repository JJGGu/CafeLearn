# 感官参考图

这个目录收集了 `SCA Coffee Taster's Flavor Wheel` 和 `WCR Sensory Lexicon` 的官方参考文件。

## 推荐优先看这些

1. `SCA_Coffee_Tasters_Flavor_Wheel_Chinese_Simplified.png`
2. `SCA_Coffee_Tasters_Flavor_Wheel_English.png`
3. `WCR_Sensory_Lexicon_v2_2025_Chinese.pdf`
4. `WCR_Sensory_Lexicon_v2_2025_English.pdf`
5. `WCR_Sensory_Lexicon_v2_2025_Chinese_preview.png`

## 说明

### Flavor Wheel

- `SCA` 官方真正的高清数字版通常在商店里以 `PDF/EPS` 形式出售。
- 这个目录里的 `PNG` 文件来自 `SCA` 官方商店公开预览图，分辨率约为 `2160 × 2160`，足够日常查看和参考。
- 简中预览图文件：
  `SCA_Coffee_Tasters_Flavor_Wheel_Chinese_Simplified.png`
- 英文预览图文件：
  `SCA_Coffee_Tasters_Flavor_Wheel_English.png`

### WCR Sensory Lexicon

- 这个目录里的 `WCR` 文件来自 `World Coffee Research` 官方下载页。
- 已下载最新版 `v2.0` 的中文和英文 PDF。
- `WCR_Sensory_Lexicon_v2_2025_Chinese_preview.png` 是本地从中文 PDF 生成的首页预览图，方便快速浏览。
