# 官方资料目录

这个目录收集公开可下载的一手资料，优先放：

1. 协会课程与白皮书。
2. 研究机构发布的品种与感官资料。
3. 赛事规则、 roast plan 与评分表。
4. 官方软件入门文档。

总入口请先看： [官方学习资料与参考下载索引.md](../../官方学习资料与参考下载索引.md)
