## 一、HiBean vs Artisan 对比分析

### 两款软件定位

| 维度 | HiBean | Artisan |
|------|--------|---------|
| **定位** | 移动端烘焙管理 App | 桌面端专业烘焙记录软件 |
| **平台** | iOS / Android | Windows / Mac / Linux |
| **价格** | 免费（部分高级功能付费） | 完全免费开源 |
| **上手难度** | ⭐⭐ 简单直观 | ⭐⭐⭐⭐ 学习曲线陡峭 |
| **界面** | 现代、美观、移动友好 | 功能强大但界面较老 |

### 功能对比

| 功能 | HiBean | Artisan |
|------|--------|---------|
| **实时曲线记录** | ✅ 支持（需蓝牙连接烘焙机） | ✅ 支持（需串口/蓝牙连接） |
| **手动记录模式** | ✅ 支持 | ✅ 支持 |
| **烘焙机兼容性** | 主要兼容 Kaleido、Skywalker 等新型小烘焙机 | 兼容几乎所有主流烘焙机（200+型号） |
| **曲线叠加对比** | ✅ 支持 | ✅ 支持，且更强大 |
| **生豆管理** | ✅ 有库存管理功能 | ❌ 无 |
| **烘焙日志管理** | ✅ 云端同步 | ✅ 本地存储 |
| **数据导出** | ✅ 支持导出 | ✅ 支持多种格式导出 |
| **RoR（升温速率）** | ✅ 实时显示 | ✅ 实时显示，可深度自定义 |
| **PID 控制** | 部分支持 | ✅ 支持 |
| **社区与资源** | 较新，社区较小 | 历史悠久，社区庞大，教程丰富 |
| **杯测记录** | ✅ 有基础杯测功能 | ❌ 无 |
| **移动端使用** | ✅ 原生 App | ❌ 仅桌面端 |

### 用户评价总结（来自 Reddit、Facebook 社群）

**HiBean 优势**：
- "比 Artisan 简单太多了，上手很快"
- "界面漂亮，视觉体验好"
- "手机操作方便，不用搬电脑"
- "生豆管理和杯测记录一体化"

**HiBean 劣势**：
- 烘焙机兼容性不如 Artisan 广
- 高级自定义功能不如 Artisan 丰富
- 社区资源和教程相对较少

**Artisan 优势**：
- 完全免费开源
- 兼容几乎所有烘焙机
- 数据分析功能极其强大
- 社区庞大，遇到问题容易找到答案

**Artisan 劣势**：
- 学习曲线非常陡峭
- 界面不够现代
- 需要电脑，不够便携

### 结论：你用 HiBean 完全可以

**对你当前阶段来说，HiBean 是更好的选择**，原因：

1. **你是新手**：HiBean 上手快，不会在软件配置上浪费时间
2. **移动端方便**：工作日晚间烘焙，手机操作比搬电脑方便
3. **一体化管理**：生豆、烘焙、杯测在一个 App 里
4. **核心功能够用**：曲线记录、RoR 显示、曲线对比这些训练必需功能都有

**什么时候考虑切换到 Artisan**：
- 当你需要更精细的数据分析（如自定义 RoR 算法）
- 当你开始做生产烘焙，需要更严格的 QC 流程
- 当你的烘焙机不被 HiBean 支持

> **建议**：现阶段专注用好 HiBean，不要在工具选择上纠结。工具只是辅助，你的感官和判断力才是核心。

## 二、哪些数据需要数字化处理

### 必须数字化的（用 HiBean + 手机/电脑）

| 数据类型 | 原因 | 工具 |
|----------|------|------|
| **烘焙曲线** | 需要叠加对比、追踪趋势 | HiBean |
| **烘焙参数记录** | 需要搜索、筛选、统计 | HiBean + 电子表格 |
| **TDS/萃取率数据** | 需要和烘焙参数关联分析 | 电子表格（Excel/Numbers/Google Sheets） |
| **生豆库存** | 需要追踪用量和剩余 | HiBean 或电子表格 |
| **杯测评分趋势** | 需要看长期进步曲线 | 电子表格 |
| **照片记录** | 豆色对比、生豆外观 | 手机相册（按日期分文件夹） |

### 建议保持纸质的（打印 + 手写）

| 数据类型 | 原因 |
|----------|------|
| **杯测记录表** | 杯测时手写更专注，不会被手机分心 |
| **感官训练记录** | 手写过程本身是记忆强化 |
| **标签** | 贴在袋子上，随时可看 |
| **每周训练总结** | 手写反思效果更好 |

### 推荐的数字化工作流

```
烘焙时 → HiBean 实时记录曲线
         ↓
烘焙后 → 手写标签贴袋 + HiBean 补充参数
         ↓
杯测时 → 纸质杯测表手写记录
         ↓
杯测后 → 关键数据录入电子表格（评分、TDS、结论）
         ↓
每周末 → 电子表格做趋势分析
```

### 电子表格模板建议

你的 `烘焙记录模板.csv` 已经更新，可以直接用 Excel/Numbers/Google Sheets 打开。

建议在电子表格中额外建立以下 Sheet：

| Sheet 名 | 内容 |
|----------|------|
| **烘焙日志** | 就是 CSV 模板的内容 |
| **杯测评分汇总** | 每炉的 6 项评分 + 总分，方便画趋势图 |
| **TDS 追踪** | 每次萃取的 TDS、萃取率、研磨度、口感评价 |
| **生豆库存** | 每支豆的购买量、已用量、剩余量 |
| **问题-修正记录** | 杯感问题 → 判断原因 → 修正动作 → 修正结果 |

## 三、TANK200D 设备详细分析与风门问题

### 你的设备：TANK200D 完整参数

| 参数 | 值 | 对训练的意义 |
|------|------|------------|
| **型号** | TANK200D 咖啡烘焙机 | — |
| **烘焙类型** | 直火 | 辐射热为主，局部过热风险需注意 |
| **烘焙容量** | 120g-220g | 训练载量建议固定 **200g** |
| **滚筒** | 不锈钢直火滚筒，单层多片拨片 | 搅拌均匀度取决于载量 |
| **滚筒转速** | 55RPM | 固定值，不可调 |
| **加热方式** | 电热（电加热管） | 比燃气更稳定，升温响应略慢 |
| **火力调节** | 无极调节按钮 | ⭐ 你最主要的控制手段 |
| **火力指示** | 实时功率显示 | 可以精确记录火力百分比 |
| **升温能力** | 1-20°C/分钟 | 范围足够训练使用 |
| **测温** | 豆温探针 | 只有豆温，无进风温/排风温 |
| **显示** | 4.3英寸触屏 | 可直接读取温度和时间 |
| **计时** | 自动计时 | 方便记录 |
| **风门** | 冷却盘配合排烟筒使用 | ⚠️ **非独立风门**，这就是你感觉风门作用不大的原因 |
| **冷却** | 风冷冷却盘 | 下豆后自动冷却 |
| **银皮收集** | 专用排烟筒可收集银皮 | 需要定期清理 |
| **排烟口直径** | 70mm | — |
| **烘焙时间** | 8-15分钟 | 正常范围 |
| **功率** | 1400W / 220V | 家用电路可承受 |
| **数据输出** | 外接群特301/华仪6514 | 可连接外部温度记录仪 |
| **尺寸** | 600×180×400mm | 桌面可放 |
| **重量** | 25KG | 固定放置，不建议频繁移动 |

### TANK200D 的风门为什么"作用不大"

看了参数后，原因很清楚了：**TANK200D 没有独立的风门调节机构**。

它的"风门"实际上是冷却盘配合排烟筒的排烟通道，主要功能是：
1. 排出烘焙烟气
2. 收集银皮
3. 提供基础通风

它**不是**像专业烘焙机那样可以精细调节气流量的独立风门。所以你感觉"风门作用不大"是完全正确的判断——因为它本来就不是设计来做精细气流控制的。

### 这会影响你的学习进步吗？

**不会严重影响**，原因如下：

1. **TANK200D 的核心控制手段是火力（无极调节）**，这已经足够你做分段降火、单变量实验等所有训练内容。

2. **电热加热比燃气更稳定**，功率显示让你可以精确记录火力百分比，这对复现性训练非常有利。

3. **很多世界级烘焙师的训练起步机器也很简单**。Scott Rao 明确说过，理解烘焙原理比设备复杂度更重要。

4. **你当前阶段（前 24 周）最重要的能力是**：
   - 感官识别能力
   - 单变量实验的逻辑
   - 杯感到过程的映射能力
   - 复现能力

   这些能力的建立**不依赖风门**。

### TANK200D 的优势（你可能没意识到的）

| 优势 | 说明 |
|------|------|
| **电热 + 功率显示** | 可以精确记录"火力 60%"这样的数值，比燃气的"火力中偏大"更可量化 |
| **无极调节** | 火力调节精度高，适合做微调实验 |
| **自动计时 + 触屏** | 减少手动记录负担 |
| **豆温探针** | 直接读取豆温，数据可靠 |
| **容量小** | 每炉只用 200g 生豆，试错成本低（每炉生豆成本约 15-25 元） |
| **数据输出接口** | 可以外接群特301/华仪6514 做更精确的温度记录 |

### TANK200D 的局限与应对

| 局限 | 影响程度 | 应对方式 |
|------|---------|---------|
| 无独立风门 | 中等 | 用火力分段控制替代；烘焙时保持通风 |
| 容量小（最大 220g） | 较小 | 训练阶段反而是优势（省豆）；生产需求时再升级 |
| 只有豆温探针 | 较小 | 豆温是最核心的数据，够用 |
| 电热升温响应略慢 | 较小 | 提前做火力调整，预判而非追赶 |
| 直火局部过热风险 | 中等 | 不要用过低载量（建议不低于 150g） |

### 是否需要换机器？

#### 现阶段（前 24 周）：不需要换

理由：
1. TANK200D 的火力控制精度（无极调节 + 功率显示）对训练来说已经很好
2. 你还没有充分挖掘这台机器的潜力
3. 换机器意味着所有基准数据重来
4. 你当前最需要提升的是感官和判断力，不是设备
5. 每炉 200g 的低成本试错是新手阶段的巨大优势

#### 什么时候考虑升级

当你满足以下 **全部** 条件时，可以考虑升级：

1. ✅ 你已经能在 TANK200D 上稳定复现同一支豆的风味
2. ✅ 你已经能做单变量实验并说清差异
3. ✅ 你已经能根据杯感提出合理修正
4. ✅ 你明确感受到 TANK200D 限制了你的进步（而不是猜测）
5. ✅ 你有明确的升级需求（比如需要更大载量、独立风门、或做生产烘焙）

#### 如果未来要升级，建议方向

| 需求 | 推荐方向 | 预算参考 |
|------|---------|---------|
| 更精细的风门 + 更大载量 | 半热风烘焙机（如 Kaleido Sniper M2/M6） | 3000-8000 元 |
| 生产能力 | 500g-1kg 级半热风/全热风 | 8000-20000 元 |
| 专业训练 + 比赛 | 专业样品烘焙机 | 15000+ 元 |

> **核心建议**：先把人练好，再升级设备。TANK200D 的电热无极调节 + 功率显示，在小型烘焙机中已经算是不错的配置了。设备是放大器，放大的是你已有的能力。

## 四、工作日/周末训练时间安排

### 你的时间约束

- 工作日：19:00 后到家
- 周末：全天可用
- 每周目标：3-4 炉烘焙 + 1 次杯测 + 感官训练

### 推荐周计划模板

| 日期 | 时间 | 活动 | 时长 |
|------|------|------|------|
| **周一** | — | 休息 | — |
| **周二晚** | 19:30-21:00 | 烘焙 2-3 炉 | 90 分钟 |
| **周三晚** | 19:30-20:30 | 杯测（如有到期样品）或感官训练 | 60 分钟 |
| **周四晚** | — | 休息或日常手冲 + TDS 测试 | 随意 |
| **周五晚** | 19:30-20:00 | 味觉溶液训练 或 香气训练 | 30 分钟 |
| **周六上午** | 9:00-12:00 | 正式杯测 + 烘焙 1-2 炉（可选） | 2-3 小时 |
| **周日上午** | 10:00-11:30 | 三角杯测训练 + 周总结 | 90 分钟 |

### 时间不够时的最小版本

如果某周特别忙，至少保证：

1. ✅ 烘焙 2 炉（周二或周六，40 分钟）
2. ✅ 杯测 1 次（周六，30 分钟）
3. ✅ 记录填写完整（10 分钟）

> 宁可少烘几炉，也不要跳过杯测和记录。没有杯测的烘焙等于没有反馈的练习。
